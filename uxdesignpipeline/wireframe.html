Interactive Tutorial

Step-by-step guide to using the Kanban board
Tutorial Steps
1 Board Overview: Learn about the three-column layout and workflow direction
2 Task Cards: Understand task information, priority levels, and assignees
3 Moving Tasks: Practice dragging tasks between columns or using keyboard shortcuts
4 Adding Tasks: Learn how to create new tasks and assign them to team members
5 Team Features: Explore collaboration tools and real-time updates
6 Mobile Usage: See how the board adapts to different screen sizes
Quick Tips
  • 💡 Use keyboard shortcuts for faster navigation
  • 💡 Drag tasks or use Enter key to move between columns
  • 💡 Click task cards to view detailed information
  • 💡 Use priority colors to identify urgent tasks
  • 💡 Check team activity feed for recent updates
