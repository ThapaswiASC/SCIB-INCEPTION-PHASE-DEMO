### 4.1 AppComponent

Component Type: Page

Mapped From A1: Root application container

Purpose: Main application shell providing layout structure and global state management

Parent: None (root component)

Children:
- SidebarNavigationComponent
- MainContentComponent

Dependencies:
- AuthService
- UserService
- Router

Library Components Used:
- None directly

TypeScript Specification

Inputs:
- None (root component)

Outputs:
- None (root component)

State:
- user: User | null
- loading: boolean
- sidebarCollapsed: boolean
- authenticated: boolean

Methods:
- ngOnInit() → Initialize application
- toggleSidebar() → Toggle sidebar visibility
- handleLogout() → User logout

HTML STRUCTURE (PSEUDO CODE ONLY)

<div class="app-container">
  <app-sidebar-navigation 
    [activeView]="activeView" 
    [collapsed]="sidebarCollapsed"
    (viewChanged)="onViewChanged($event)"
    (menuToggle)="toggleSidebar()">
  </app-sidebar-navigation>
  
  <app-main-content 
    [user]="user" 
    [loading]="loading"
    (searchChanged)="onSearchChanged($event)">
  </app-main-content>
</div>

CSS SPECIFICATION

Layout:
- display: grid
- grid-template-columns: 280px 1fr
- height: 100vh

Spacing:
- gap: 0
- margin: 0
- padding: 0

Component Styles:
- .app-container: main grid container
- .sidebar-collapsed: reduced sidebar width

Responsive:
- Mobile (<768px): stack layout, collapsible sidebar
- Tablet (768px-1024px): reduced sidebar width
- Desktop (>1024px): full layout

API INTEGRATION

- ngOnInit() → GET /users/profile → Load user data
- handleLogout() → POST /auth/logout → User logout

ERROR HANDLING

- Authentication failure → redirect to login
- API failure → show global error message
- Network error → show offline indicator

INTERACTION FLOW

User Action → UI → API
- App load → initialize components → GET /users/profile
- Sidebar toggle → update layout → no API call
- Logout → confirm dialog → POST /auth/logout

### 4.4 KanbanBoardComponent

Component Type: Feature

Mapped From A1: Main container for three-column kanban layout

Purpose: Core kanban board functionality with drag-and-drop and task management

Parent: ContentAreaComponent

Children:
- KanbanColumnComponent (x3)
- TaskModalComponent

Dependencies:
- TaskService
- KanbanService
- DragDropModule

Library Components Used:
- Button Component → Create task button
- Modal Component → Task creation/editing

TypeScript Specification

Inputs:
- tasks: Task[]
- loading: boolean
- error: string | null

Outputs:
- taskCreated: EventEmitter<Task>
- taskUpdated: EventEmitter<Task>
- taskDeleted: EventEmitter<string>
- taskMoved: EventEmitter<{taskId: string, columnId: string}>

State:
- columns: KanbanColumn[]
- selectedTask: Task | null
- modalOpen: boolean
- draggedTask: Task | null
- modalMode: 'create' | 'edit'

Methods:
- ngOnInit() → Initialize columns and load tasks
- openCreateModal() → Open task creation modal
- openEditModal(task: Task) → Open task editing modal
- onTaskDropped(event: CdkDragDrop<Task[]>) → Handle drag-and-drop
- onTaskCreated(task: Task) → Handle new task creation
- onTaskUpdated(task: Task) → Handle task updates
- onTaskDeleted(taskId: string) → Handle task deletion

HTML STRUCTURE (PSEUDO CODE ONLY)

<div class="kanban-board" 
     role="region" 
     aria-label="Kanban Board"
     [class.loading]="loading">
  
  <div class="board-header">
    <div class="board-title">
      <h1>Task Management</h1>
      <p>Organize and track your team's work</p>
    </div>
    <aava-button 
      variant="primary"
      (click)="openCreateModal()">
      Create Task
    </aava-button>
  </div>
  
  <div class="board-columns" 
       cdkDropListGroup>
    <app-kanban-column 
      *ngFor="let column of columns; trackBy: trackByColumnId"
      [column]="column"
      [tasks]="getTasksForColumn(column.id)"
      [allowDrop]="!loading"
      (taskDropped)="onTaskDropped($event)"
      (taskSelected)="openEditModal($event)"
      (taskDeleted)="onTaskDeleted($event)">
    </app-kanban-column>
  </div>
  
  <app-task-modal 
    [task]="selectedTask"
    [visible]="modalOpen"
    [mode]="modalMode"
    (taskSaved)="onTaskSaved($event)"
    (modalClosed)="closeModal()">
  </app-task-modal>
  
  <div class="error-state" *ngIf="error">
    <p>{{error}}</p>
    <aava-button (click)="retryLoad()">Retry</aava-button>
  </div>
</div>

CSS SPECIFICATION

Layout:
- display: flex
- flex-direction: column
- height: 100%
- gap: 1.5rem

Spacing:
- padding: 1.5rem
- .board-columns: display: grid, grid-template-columns: repeat(3, 1fr), gap: 1.5rem

Component Styles:
- .kanban-board: main container
- .board-header: flex layout, justify-content: space-between
- .board-columns: grid layout for columns
- .loading: opacity: 0.6, pointer-events: none

Responsive:
- Mobile (<768px): single column stack
- Tablet (768px-1024px): two columns
- Desktop (>1024px): three columns

API INTEGRATION

- ngOnInit() → GET /tasks → Fetch all tasks
- onTaskCreated() → POST /tasks → Create new task
- onTaskUpdated() → PUT /tasks/:id → Update task
- onTaskDeleted() → DELETE /tasks/:id → Delete task
- onTaskDropped() → PUT /tasks/:id/column → Move task between columns

ERROR HANDLING

- API failure → show error message with retry button
- Validation error → show inline form errors
- Network error → show offline indicator
- Drag-and-drop failure → revert task position

INTERACTION FLOW

User Action → UI → API
- Load board → show loading → GET /tasks
- Create task → open modal → fill form → POST /tasks
- Edit task → click card → open modal → PUT /tasks/:id
- Delete task → confirm dialog → DELETE /tasks/:id
- Move task → drag and drop → PUT /tasks/:id/column

        <button (click)="addTask()">Add Task</button>

    </aava-popover>
  </div>

       cdkDropList 
       [cdkDropListData]="tasks"
       [cdkDropListDisabled]="!allowDrop"
       (cdkDropListDropped)="onTaskDrop($event)">
    
    <app-task-card 
      *ngFor="let task of tasks; trackBy: trackByTaskId"
      [task]="task"
      [draggable]="allowDrop"
      cdkDrag

      (taskDeleted)="onTaskDeleted(task.id)">
    </app-task-card>
    
    <div class="empty-state" *ngIf="tasks.length === 0">
      <p>No tasks in {{column.title.toLowerCase()}}</p>
      <aava-button variant="ghost" (click)="addTask()">Add First Task</aava-button>

  </div>
