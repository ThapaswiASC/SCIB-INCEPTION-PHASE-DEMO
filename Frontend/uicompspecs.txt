### 4.1 AppComponent

Component Type: Page

Mapped From A1: Root application component

Purpose: Main application container with layout structure

Parent: None (root)

Children:
- HeaderComponent
- SidebarComponent
- KanbanBoardComponent

Dependencies:
- Angular Core
- Router

Outputs:
- None

State:
- user: User

Methods:

- toggleSidebar() → Toggle sidebar visibility

HTML STRUCTURE (PSEUDO CODE ONLY)

<div class="app-container">
  <app-sidebar 
    [activeRoute]="currentRoute" 
    [isCollapsed]="sidebarCollapsed"
    (onNavigate)="handleNavigation($event)">

  <div class="main-layout">
    <app-header 
      [user]="user" 
      [notifications]="notifications"
      (onSearch)="handleSearch($event)">

      <app-kanban-board 
        [columns]="kanbanColumns" 
        [tasks]="tasks" 
        [loading]="isLoading">
      </app-kanban-board>
    </main>
  </div>
</div>

- display: flex
- height: 100vh

Spacing:
- margin: 0
- padding: 0

- .content-area: flex-grow 1

- Mobile: sidebar overlay
- Tablet: collapsible sidebar
- Desktop: full sidebar

- handleAuth() → GET /api/youth-accounts/{id}/validate → User authentication

ERROR HANDLING

- API failure → show error toast
- Authentication error → redirect to login

INTERACTION FLOW

User Action → UI → API
- App Load → ngOnInit() → fetch initial data → render components

### 4.4 KanbanBoardComponent

Component Type: Feature

Mapped From A1: Main kanban board container

Purpose: Container for three-column kanban layout with drag-and-drop

Parent: AppComponent

Children:
- BoardControlsComponent
- KanbanColumnComponent (3 instances)
- TaskModalComponent

Dependencies:
- KanbanService
- CDK Drag Drop

Library Components Used:
- Card → column containers
- Button → action buttons
- Modal → task creation

TypeScript Specification

Inputs:
- columns: Array<KanbanColumn>
- tasks: Array<Task>
- loading: boolean

Outputs:
- onTaskMove: EventEmitter<{taskId: string, targetColumn: string}>
- onTaskCreate: EventEmitter<Task>
- onFilterChange: EventEmitter<FilterState>

State:
- selectedFilters: FilterState
- draggedTask: Task | null
- showTaskModal: boolean
- modalTargetColumn: string

Methods:
- ngOnInit() → Initialize board data
- onTaskDrop(event: CdkDragDrop) → Handle task movement
- openTaskModal(columnId: string) → Open task creation modal
- handleTaskCreate(task: Task) → Create new task

HTML STRUCTURE (PSEUDO CODE ONLY)

<div class="kanban-board-container">
  <div class="page-header">
    <nav class="breadcrumb">
      <span>Boards</span> / <span>My Kanban Board</span>
    </nav>
    
    <div class="page-title-section">
      <h1>Task Management Board</h1>
      <p>Organize and track your tasks efficiently</p>
    </div>
    
    <div class="page-actions">
      <app-button variant="primary" (click)="openTaskModal('todo')">
        Add Task
      </app-button>
      <app-button variant="secondary" (click)="exportReport()">
        Export Report
      </app-button>
    </div>
  </div>
  
  <app-board-controls 
    [filters]="selectedFilters"
    (onFilterChange)="handleFilterChange($event)">
  </app-board-controls>
  
  <div class="kanban-container" 
       cdkDropListGroup
       [class.loading]="loading">
    
    <app-kanban-column 
      *ngFor="let column of columns" 
      [column]="column" 
      [tasks]="getTasksForColumn(column.id)"
      [allowDrop]="true"
      (onTaskDrop)="onTaskDrop($event)"
      (onAddTask)="openTaskModal(column.id)">
    </app-kanban-column>
  </div>
  
  <app-task-modal 
    [isOpen]="showTaskModal"
    [task]="null"
    [targetColumn]="modalTargetColumn"
    (onSubmit)="handleTaskCreate($event)"
    (onClose)="closeTaskModal()">
  </app-task-modal>
</div>

CSS SPECIFICATION

Layout:
- display: flex
- flex-direction: column
- height: 100%

Spacing:
- padding: 2rem
- gap: 1.5rem

Component Styles:
- .kanban-container: display grid, grid-template-columns 1fr 1fr 1fr
- .page-header: flex layout with space-between
- .loading: opacity and pointer-events

Responsive:
- Mobile: single column, tabs for switching
- Tablet: two columns
- Desktop: three columns

API INTEGRATION

- ngOnInit() → GET /api/youth-accounts/{id}/dashboard → Load board data
- handleTaskCreate() → POST /api/youth-accounts/{id}/fund-transfer → Create task
- onTaskMove() → PUT /api/youth-accounts/{id}/spending-limit → Update task status

ERROR HANDLING

- API failure → show error snackbar
- Drag drop error → revert task position

INTERACTION FLOW

User Action → UI → API
- Load Board → ngOnInit() → fetch data → render columns and tasks
- Drag Task → onTaskDrop() → update state → API call → refresh data
- Add Task → openTaskModal() → show modal → submit → API call → refresh

### 4.5 KanbanColumnComponent

Component Type: Feature

Mapped From A1: Individual column container

Purpose: Container for tasks in a specific status column

Parent: KanbanBoardComponent

Children:
- TaskCardComponent (multiple)

Dependencies:
- CDK Drag Drop

Library Components Used:
- Card → column container
- Button → add task button

TypeScript Specification

Inputs:
- column: KanbanColumn
- tasks: Array<Task>
- allowDrop: boolean

Outputs:
- onTaskDrop: EventEmitter<CdkDragDrop<Task[]>>
- onAddTask: EventEmitter<void>

State:
- isDragOver: boolean

Methods:
- drop(event: CdkDragDrop) → Handle task drop
- dragEnter() → Set drag over state
- dragLeave() → Clear drag over state

HTML STRUCTURE (PSEUDO CODE ONLY)

<div class="kanban-column" 
     [attr.aria-label]="column.title + ' column'">
  
  <div class="column-header">
    <h2 class="column-title">{{ column.title }}</h2>
    <span class="task-count">{{ tasks.length }}</span>
    
    <button class="add-task-btn" 
            (click)="onAddTask.emit()"
            [attr.aria-label]="'Add task to ' + column.title">
      <i class="icon-plus"></i>
    </button>
  </div>
  
  <div class="column-content" 
       cdkDropList
       [cdkDropListData]="tasks"
       [cdkDropListDisabled]="!allowDrop"
       (cdkDropListDropped)="drop($event)"
       (cdkDropListEntered)="dragEnter()"
       (cdkDropListExited)="dragLeave()"
       [class.drag-over]="isDragOver">
    
    <app-task-card 
      *ngFor="let task of tasks; trackBy: trackByTaskId" 
      [task]="task" 
      [draggable]="true"
      cdkDrag
      [cdkDragData]="task">
    </app-task-card>
    
    <div class="empty-state" *ngIf="tasks.length === 0">
      <p>No tasks in {{ column.title.toLowerCase() }}</p>
      <button class="add-first-task" (click)="onAddTask.emit()">
        Add your first task
      </button>
    </div>
  </div>
</div>

CSS SPECIFICATION

Layout:
- display: flex
- flex-direction: column
- min-height: 500px

Spacing:
- padding: 1rem
- gap: 1rem

Component Styles:
- .kanban-column: border, border-radius, background
- .column-header: flex layout with space-between
- .drag-over: border color change, background highlight

Responsive:
- Mobile: full width
- Tablet: flex-basis 50%
- Desktop: flex-basis 33.33%

API INTEGRATION

- None (receives data from parent)

ERROR HANDLING

- Drop error → show error message
- Invalid drop → revert to original position

INTERACTION FLOW

User Action → UI → API
- Drop Task → drop() → emit event → parent handles API call
- Add Task → click button → emit event → parent opens modal
