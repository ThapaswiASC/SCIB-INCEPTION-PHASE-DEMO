            <h2>User Flow Overview</h2>
            <p>This diagram illustrates the primary user interaction flows within the Kanban Board Management System. The flow covers user authentication, task management, navigation, and key feature interactions across the three-column layout (To Do, In Progress, Done).</p>

        <div class="mermaid">
            flowchart TD
                Start[User Opens Application]
                Start --> LoadApp[Load Application]
                LoadApp --> CheckAuth{User Authenticated?}
                CheckAuth -->|No| Login[Show Login Screen]
                Login --> AuthAPI[Call Auth API]
                AuthAPI --> Dashboard[Load Dashboard]
                CheckAuth -->|Yes| Dashboard
                
                Dashboard --> LoadProfile[Load User Profile]
                LoadProfile --> LoadTasks[Load Kanban Board]
                LoadTasks --> DisplayBoard[Display Three Column Board]
                
                DisplayBoard --> UserActions{User Action}
                
                UserActions --> CreateTask[Click Create Task Button]
                CreateTask --> OpenModal[Open Task Creation Modal]
                OpenModal --> FillForm[Fill Task Form]
                FillForm --> SubmitTask[Submit New Task]
                SubmitTask --> SaveTask[Save Task via API]
                SaveTask --> RefreshBoard[Refresh Kanban Board]
                RefreshBoard --> DisplayBoard
                
                UserActions --> ViewTask[Click Task Card]
                ViewTask --> ShowDetails[Show Task Details]
                ShowDetails --> EditTask{Edit Task?}
                EditTask -->|Yes| OpenEditModal[Open Edit Modal]
                OpenEditModal --> UpdateForm[Update Task Form]
                UpdateForm --> SaveChanges[Save Changes via API]
                SaveChanges --> RefreshBoard
                EditTask -->|No| DisplayBoard
                
                UserActions --> DragTask[Drag Task Card]
                DragTask --> DropColumn[Drop in New Column]
                DropColumn --> UpdateStatus[Update Task Status]
                UpdateStatus --> SaveMove[Save Move via API]
                SaveMove --> RefreshBoard
                
                UserActions --> SearchTasks[Use Search Bar]
                SearchTasks --> FilterResults[Filter Task Results]
                FilterResults --> DisplayFiltered[Display Filtered Tasks]
                DisplayFiltered --> ClearSearch{Clear Search?}
                ClearSearch -->|Yes| DisplayBoard
                ClearSearch -->|No| DisplayFiltered
                
                UserActions --> NavigateMenu[Use Sidebar Navigation]
                NavigateMenu --> ChangeView[Change Workflow View]
                ChangeView --> LoadNewView[Load New View Data]
                LoadNewView --> DisplayBoard
                
                UserActions --> UserProfile[Click User Avatar]
                UserProfile --> ShowProfileMenu[Show Profile Menu]
                ShowProfileMenu --> ProfileActions{Profile Action}
                ProfileActions --> ViewProfile[View Profile]
                ProfileActions --> Settings[Access Settings]
                ProfileActions --> Logout[Logout]
                ViewProfile --> DisplayBoard
                Settings --> DisplayBoard
                Logout --> Start
                
                UserActions --> ErrorState[Error Occurred]
                ErrorState --> ShowError[Display Error Message]
                ShowError --> RetryAction{Retry Action?}
                RetryAction -->|Yes| UserActions
                RetryAction -->|No| DisplayBoard
        </div>

            <h2>Key User Interaction Points</h2>
            <p><strong>Primary Flows:</strong></p>
            <ul>
                <li><strong>Task Creation:</strong> Create Task Button → Modal Form → API Save → Board Refresh</li>
                <li><strong>Task Management:</strong> Task Card Click → View/Edit → Update → Save Changes</li>
                <li><strong>Drag & Drop:</strong> Drag Task → Drop in Column → Status Update → API Save</li>
                <li><strong>Search & Filter:</strong> Search Bar → Filter Results → Display Filtered View</li>
                <li><strong>Navigation:</strong> Sidebar Menu → Change View → Load Data → Display Board</li>
                <li><strong>User Management:</strong> Profile Avatar → Menu → Profile/Settings/Logout</li>
            </ul>
            <p><strong>Error Handling:</strong> All actions include error states with retry mechanisms and user feedback.</p>
