        <div class="description">
            <h2>Key User Flows Explained</h2>
            <p><strong>Primary Flow:</strong> Users start by viewing the three-column Kanban board (To Do, In Progress, Done) and can interact with task cards through drag-and-drop, creation, editing, and detailed viewing.</p>
            <p><strong>Navigation Flow:</strong> The sidebar provides access to different application modules including Board, Settings, Analytics, and Reports, each with their own sub-flows.</p>
            <p><strong>Task Management Flow:</strong> Users can create new tasks through modals, edit existing tasks, move tasks between columns via drag-and-drop, and perform bulk operations on multiple tasks.</p>
            <p><strong>Settings Flow:</strong> Users can customize display preferences, accessibility options, notifications, and account settings with persistent storage.</p>
            <p><strong>Analytics Flow:</strong> Users can view performance statistics, charts, and activity feeds to track productivity and team performance.</p>
            <p><strong>Reports Flow:</strong> Users can configure, generate, preview, and download custom reports based on task data and team metrics.</p>
