            <p><strong>Authentication Flow:</strong> Users start by logging into the application, with error handling for failed attempts.</p>

            <p><strong>Task Management:</strong> Users can view, create, edit, and delete tasks across three Kanban columns (To Do, In Progress, Done).</p>

            <p><strong>File Management:</strong> Users can upload and download task attachments with document validation.</p>
            <p><strong>Search and Filter:</strong> The header search allows users to filter tasks across all columns.</p>
