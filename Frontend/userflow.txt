                Start[User Opens Application]

                AuthSuccess -->|No| LoginError[Show Login Error]
                LoginError --> Login
                AuthSuccess -->|Yes| Dashboard
                
                Dashboard --> LoadProfile[GET /users/profile]
                LoadProfile --> LoadBoard[Load Kanban Board]
                LoadBoard --> GetBoardData[GET /forms/sessions/sessionId]

                RenderBoard --> TodoColumn[To Do Column]

                RenderBoard --> DoneColumn[Done Column]
                
                TodoColumn --> ViewTasks[Display Task Cards]
                InProgressColumn --> ViewTasks
                DoneColumn --> ViewTasks
                
                ViewTasks --> UserActions{User Action?}
                
                UserActions -->|Click Add Task| CreateTask[Open Task Creation Modal]
                UserActions -->|Click Task Card| ViewTaskDetail[Open Task Detail Modal]
                UserActions -->|Drag Task| DragTask[Start Drag Operation]
                UserActions -->|Use Search| SearchTasks[Filter Tasks]
                UserActions -->|Click Profile| UserProfile[Open Profile Menu]
                UserActions -->|Click Sidebar| Navigate[Navigate to Section]
                
                CreateTask --> TaskForm[Fill Task Form]
                TaskForm --> HasAttachment{Has Attachment?}
                HasAttachment -->|Yes| UploadDoc[POST /documents/upload]
                HasAttachment -->|No| SaveTask[PUT /forms/sessions/sessionId]
                UploadDoc --> SaveTask
                SaveTask --> TaskCreated[Task Created Successfully]
                TaskCreated --> RefreshBoard[Refresh Board State]
                RefreshBoard --> RenderBoard
                
                ViewTaskDetail --> EditTask{Edit Task?}
                EditTask -->|Yes| UpdateTask[Modify Task Data]
                EditTask -->|No| CloseModal[Close Modal]
                UpdateTask --> SaveChanges[PUT /forms/sessions/sessionId]
                SaveChanges --> RefreshBoard
                CloseModal --> ViewTasks
                
                DragTask --> DropZone{Valid Drop Zone?}
                DropZone -->|Yes| MoveTask[Update Task Position]
                DropZone -->|No| CancelDrag[Cancel Drag Operation]
                MoveTask --> UpdatePosition[PUT /forms/sessions/sessionId]
                UpdatePosition --> RefreshBoard
                CancelDrag --> ViewTasks
                
                SearchTasks --> FilterResults[Show Filtered Results]
                FilterResults --> ViewTasks

            <h2>Key User Interactions</h2>
            <p><strong>Authentication Flow:</strong> Users start by logging in, which triggers API authentication and profile loading.</p>
            <p><strong>Board Management:</strong> The main kanban board displays three columns (To Do, In Progress, Done) with task cards that users can interact with.</p>
            <p><strong>Task Operations:</strong> Users can create new tasks, edit existing ones, and move tasks between columns using drag-and-drop functionality.</p>
            <p><strong>Navigation:</strong> The sidebar provides access to different views including workspace management, mobile views, team collaboration, and analytics.</p>
            <p><strong>Search & Filter:</strong> Users can search for specific tasks using the header search functionality.</p>
            <p><strong>Profile Management:</strong> Users can access their profile, settings, and logout functionality through the header menu.</p>
