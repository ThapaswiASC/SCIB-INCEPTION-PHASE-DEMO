### 8. API SUMMARY

**Available APIs:**
- **GET /api/youth-accounts/{youthAccountId}/dashboard** → Fetch dashboard data including balance, spending limit, recent transactions
- **POST /api/youth-accounts/{youthAccountId}/fund-transfer** → Transfer funds to youth account
- **PUT /api/youth-accounts/{youthAccountId}/spending-limit** → Configure weekly spending limits
- **GET /api/youth-accounts/{youthAccountId}/transactions** → Get paginated transaction history with filtering

**Authentication:**
- OAuth 2.0 with JWT tokens (1-hour expiration with refresh)
- Required scopes: youth-account:read, youth-account:write, fund-transfer:execute
- Rate limiting: 100 requests per minute per token

**Response Models:**
- DashboardResponse: account balance, spending limit, recent transactions
- FundTransferResponse: transfer status, updated balance, confirmation
- SpendingLimitResponse: updated limits and effective dates
- TransactionHistoryResponse: paginated transaction list with summaries
