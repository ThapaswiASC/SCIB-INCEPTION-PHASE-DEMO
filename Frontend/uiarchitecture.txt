8. API SUMMARY

Available APIs:
- GET /api/youth-accounts/{youthAccountId}/dashboard → Fetch dashboard data with balance, spending limit, recent transactions
- POST /api/youth-accounts/{youthAccountId}/fund-transfer → Transfer funds from parent to youth account
- PUT /api/youth-accounts/{youthAccountId}/spending-limit → Configure weekly spending limits
- GET /api/youth-accounts/{youthAccountId}/transactions → Get paginated transaction history with filtering

Authentication:
- OAuth 2.0 with JWT tokens
- Scopes: youth-account:read, youth-account:write, fund-transfer:execute
- Rate limiting: 100 requests per minute

9. API TO UI MAPPING

Component → API:
- DashboardPageComponent → GET /api/youth-accounts/{youthAccountId}/dashboard
- KanbanBoardComponent → GET /api/youth-accounts/{youthAccountId}/dashboard (for initial data)
- FundTransferModal → POST /api/youth-accounts/{youthAccountId}/fund-transfer
- SpendingLimitModal → PUT /api/youth-accounts/{youthAccountId}/spending-limit
- TransactionHistoryComponent → GET /api/youth-accounts/{youthAccountId}/transactions
- TaskCardComponent → Custom task management APIs (not in OpenAPI spec)

13. INTERACTION FLOWS

User Action → UI → API:

1. View Dashboard
   → Load DashboardPageComponent
   → Call GET /api/youth-accounts/{id}/dashboard
   → Display balance, spending limit, recent transactions

2. Transfer Funds
   → Click "Add Funds" button
   → Open FundTransferModal
   → Submit form
   → Call POST /api/youth-accounts/{id}/fund-transfer
   → Update dashboard balance
   → Show success notification

3. Configure Spending Limit
   → Click "Set Limit" button
   → Open SpendingLimitModal
   → Submit form
   → Call PUT /api/youth-accounts/{id}/spending-limit
   → Update dashboard display
   → Show confirmation

4. View Transaction History
   → Navigate to transactions page
   → Call GET /api/youth-accounts/{id}/transactions
   → Display paginated results
   → Apply filters as needed

5. Manage Kanban Tasks
   → Drag task between columns
   → Update task status locally
   → Call task update API (custom)
   → Sync with backend

6. Create New Task
   → Click "Add Task" button
   → Open TaskModal
   → Submit form
   → Call task creation API (custom)
   → Add to appropriate column
