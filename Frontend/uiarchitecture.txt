### 6. COMPONENT DEFINITIONS

**KanbanBoardComponent:**
- Purpose: Main container for the three-column Kanban layout
- Props:
  - columns: KanbanColumn[]
  - loading: boolean
  - error: string | null
- State:
  - draggedTask: Task | null
  - columnData: ColumnData[]
- Events:
  - taskMoved: EventEmitter<TaskMoveEvent>
  - taskSelected: EventEmitter<Task>
- API Binding:
  - GET /kanban/boards/{boardId}
  - PUT /kanban/tasks/{taskId}/move

**KanbanColumnComponent:**
- Purpose: Individual column container (To Do, In Progress, Done)
- Props:
  - columnId: string
  - title: string
  - tasks: Task[]
  - indicator: string
  - allowDrop: boolean
- State:
  - isDragOver: boolean
  - taskCount: number
- Events:
  - taskDropped: EventEmitter<DropEvent>
  - addTaskClicked: EventEmitter<string>
- API Binding: None (data flows from parent)

**TaskCardComponent:**
- Purpose: Individual task display with drag/drop functionality
- Props:
  - task: Task
  - draggable: boolean
  - priority: 'high' | 'medium' | 'low'
- State:
  - isDragging: boolean
- Events:
  - taskClicked: EventEmitter<Task>
  - dragStarted: EventEmitter<Task>
- API Binding: None (data flows from parent)

**HeaderComponent:**
- Purpose: Top navigation and search functionality
- Props:
  - user: User
  - notifications: Notification[]
- State:
  - searchQuery: string
  - showNotifications: boolean
- Events:
  - searchSubmitted: EventEmitter<string>
  - notificationClicked: EventEmitter<Notification>
- API Binding:
  - GET /search
  - GET /notifications

**SidebarComponent:**
- Purpose: Left navigation menu
- Props:
  - activeRoute: string
  - navigationItems: NavItem[]
- State:
  - isCollapsed: boolean
- Events:
  - routeChanged: EventEmitter<string>
- API Binding: None (static navigation)
