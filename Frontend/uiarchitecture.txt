### 6. COMPONENT DEFINITIONS

**KanbanBoardComponent:**
- **Purpose**: Main container for three-column kanban layout
- **Props**: columns (array), dragEnabled (boolean), responsive (boolean)
- **State**: columnData, draggedTask, errorState
- **Events**: taskMoved, columnUpdated, errorOccurred
- **API Binding**: GET /api/youth-accounts/{id}/dashboard for task data

**KanbanColumnComponent:**
- **Purpose**: Individual column container (To Do, In Progress, Done)
- **Props**: columnId (string), title (string), tasks (array), allowDrop (boolean)
- **State**: taskCount, isDropTarget
- **Events**: taskDropped, taskAdded, columnAction
- **API Binding**: None (receives data from parent)

**TaskCardComponent:**
- **Purpose**: Individual task display with drag-and-drop
- **Props**: task (object), draggable (boolean), priority (string)
- **State**: isDragging, isSelected
- **Events**: taskClicked, taskDragStart, taskDragEnd
- **API Binding**: None (receives data from parent)

**HeaderComponent:**
- **Purpose**: Top navigation with search and user menu
- **Props**: user (object), notifications (array)
- **State**: searchQuery, menuOpen
- **Events**: searchSubmitted, themeToggled, userMenuClicked
- **API Binding**: Search functionality (if implemented)

**SidebarNavigationComponent:**
- **Purpose**: Left navigation menu
- **Props**: currentPage (string), navigationItems (array)
- **State**: isOpen, activeItem
- **Events**: navigationChanged, sidebarToggled
- **API Binding**: None (static navigation)

**TaskFormComponent:**
- **Purpose**: Task creation and editing form
- **Props**: task (object), mode (string)
- **State**: formData, validationErrors, isSubmitting
- **Events**: taskCreated, taskUpdated, formCancelled
- **API Binding**: POST /api/youth-accounts/{id}/fund-transfer for task creation

**ButtonComponent (Library):**
- **Purpose**: Reusable button with multiple variants
- **Props**: label, variant, size, disabled, processing, iconName
- **State**: hover, active, processing
- **Events**: userClick
- **API Binding**: None (pure UI component)

**SearchBarComponent (Library):**
- **Purpose**: Search input with integrated functionality
- **Props**: placeholder, size, searchIconColor, sendButton
- **State**: searchValue, isFocused
- **Events**: searchClick, searchChange, onClose
- **API Binding**: Search API (if implemented)

### 8. API SUMMARY

**Available APIs:**
- **GET /api/youth-accounts/{youthAccountId}/dashboard** → Fetch dashboard data including account balance, spending limit, and recent transactions
- **POST /api/youth-accounts/{youthAccountId}/fund-transfer** → Transfer funds to youth account (can be repurposed for task creation)
- **PUT /api/youth-accounts/{youthAccountId}/spending-limit** → Configure spending limits (can be repurposed for task updates)
- **GET /api/youth-accounts/{youthAccountId}/transactions** → Get transaction history with pagination (can be repurposed for task history)

### 9. API TO UI MAPPING

**Component → API:**
- **KanbanBoardComponent** → GET /api/youth-accounts/{id}/dashboard (fetch initial task data)
- **TaskFormComponent** → POST /api/youth-accounts/{id}/fund-transfer (create new tasks)
- **TaskCardComponent** → PUT /api/youth-accounts/{id}/spending-limit (update task status/details)
- **DashboardPageComponent** → GET /api/youth-accounts/{id}/transactions (fetch activity data)

### 13. INTERACTION FLOWS

**User Action → UI → API:**

**Create New Task:**
- Click "Add Task" button → Open TaskFormComponent modal → Fill form → Submit → Call POST /api/youth-accounts/{id}/fund-transfer → Update KanbanBoardComponent → Close modal

**Move Task Between Columns:**
- Drag TaskCardComponent → Drop in KanbanColumnComponent → Update local state → Call PUT /api/youth-accounts/{id}/spending-limit → Update UI with response

**Search Tasks:**
- Type in SearchBarComponent → Emit searchChange event → Filter local task data → Update KanbanBoardComponent display

**Toggle Theme:**
- Click theme button in HeaderComponent → Call ThemeToggleComponent → Update CSS variables → Store preference

**View Dashboard:**
- Click "Dashboard" in SidebarNavigationComponent → Navigate to DashboardPageComponent → Call GET /api/youth-accounts/{id}/dashboard → Display data

**Mobile Column Navigation:**
- Click tab in MobileTabsComponent → Show/hide KanbanColumnComponent → Update active state
