### 6. COMPONENT DEFINITIONS

**KanbanBoardComponent:**
- Purpose: Main container for three-column Kanban layout
- Props: columns (array), tasks (array), loading (boolean), error (string)
- State: draggedTask, columnData, isLoading, errorMessage
- Events: taskMoved, taskCreated, taskUpdated, taskDeleted
- API Binding: GET /youth-accounts/{id}/dashboard for initial data

**KanbanColumnComponent:**
- Purpose: Individual column container (To Do, In Progress, Done)
- Props: columnId (string), title (string), tasks (array), count (number)
- State: isDragOver, isLoading
- Events: taskDropped, addTaskClicked, menuToggled
- API Binding: None (receives data from parent)

**TaskCardComponent:**
- Purpose: Individual task display with metadata
- Props: task (object), draggable (boolean), priority (string)
- State: isDragging, isSelected
- Events: taskClicked, dragStarted, dragEnded
- API Binding: None (receives data from parent)

**TaskModalComponent:**
- Purpose: Task creation and editing form
- Props: task (object), isOpen (boolean), mode (create/edit)
- State: formData, isSubmitting, validationErrors
- Events: taskSaved, modalClosed, formChanged
- API Binding: POST/PUT for task operations

**HeaderComponent:**
- Purpose: Top navigation with search and user actions
- Props: user (object), notifications (array)
- State: searchQuery, isUserMenuOpen, notificationCount
- Events: searchSubmitted, userMenuToggled, notificationClicked
- API Binding: GET for user data and notifications

**SidebarComponent:**
- Purpose: Left navigation menu
- Props: activeRoute (string), navigationItems (array)
- State: isCollapsed, activeItem
- Events: navigationClicked, sidebarToggled
- API Binding: None (static navigation)

**ButtonComponent (from library):**
- Purpose: Reusable button with variants and states
- Props: label, variant, size, disabled, processing, iconName
- State: isPressed, isFocused
- Events: userClick
- API Binding: None
