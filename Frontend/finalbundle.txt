## 3. UI COMPONENT SPECIFICATIONS (FROM AGENT-2)

### 3.1 KanbanBoardComponent

**Component Type:** Page
**Purpose:** Main container component for the Kanban board with three-column layout
**Parent:** AppComponent
**Children:** HeaderComponent, SidebarComponent, KanbanColumnComponent (x3), TaskModalComponent, BoardControlsComponent

**Dependencies:**
- @angular/core
- @angular/cdk/drag-drop
- KanbanService
- TaskService

**Library Components Used:**
- Card → Container wrapper
- Button → Action buttons

**TypeScript Specification:**

**Inputs:**
- youthAccountId: string
- initialFilters: FilterOptions

**Outputs:**
- taskCreated: EventEmitter<Task> (payload: Task object)
- taskMoved: EventEmitter<TaskMoveEvent> (payload: {taskId, fromColumn, toColumn})
- filterChanged: EventEmitter<FilterOptions> (payload: FilterOptions object)

**State:**
- columns: Column[] (array of column objects)
- tasks: Task[] (array of task objects)
- draggedTask: Task | null
- selectedFilters: FilterOptions
- loading: boolean
- error: string | null
- isModalOpen: boolean
- selectedTask: Task | null

**Methods:**
- ngOnInit() → Initialize component and load data
- loadBoardData() → Fetch dashboard and transaction data
- onTaskMove(event: CdkDragDrop<Task[]>) → Handle drag and drop
- openTaskModal(task?: Task) → Open task creation/edit modal
- closeTaskModal() → Close modal and reset state
- onFilterChange(filters: FilterOptions) → Update filters and reload data
- handleError(error: any) → Process and display errors

**HTML STRUCTURE:**

```html
<div class="kanban-board-container" [attr.aria-label]="'Kanban Board'">
  <app-header 
    [user]="currentUser" 
    [notifications]="notifications"
    (onSearch)="handleSearch($event)">
  </app-header>
  
  <div class="main-layout">
    <app-sidebar 
      [menuItems]="navigationItems"
      [activeItem]="activeRoute"
      (onMenuItemClick)="handleNavigation($event)">
    </app-sidebar>
    
    <main class="content-area" role="main">
      <div class="page-header">
        <app-breadcrumbs [items]="breadcrumbItems"></app-breadcrumbs>
        <div class="title-section">
          <h1>Kanban Board</h1>
          <p>Manage your tasks efficiently</p>
        </div>
        <div class="action-buttons">
          <app-button variant="primary" (click)="openTaskModal()">Add Task</app-button>
          <app-button variant="secondary">Export Report</app-button>
        </div>
      </div>
      
      <app-board-controls 
        [filters]="selectedFilters"
        (filterChange)="onFilterChange($event)">
      </app-board-controls>
      
      <div class="kanban-container" 
           cdkDropListGroup 
           [attr.aria-label]="'Task Board'">
        <app-kanban-column 
          *ngFor="let column of columns; trackBy: trackByColumnId"
          [column]="column"
          [tasks]="getTasksForColumn(column.id)"
          (taskMoved)="onTaskMove($event)"
          (taskClicked)="openTaskModal($event)">
        </app-kanban-column>
      </div>
      
      <div *ngIf="error" class="error-state" role="alert">
        <p>{{ error }}</p>
        <app-button (click)="loadBoardData()">Retry</app-button>
      </div>
    </main>
  </div>
  
  <app-task-modal 
    [isOpen]="isModalOpen"
    [task]="selectedTask"
    (taskSubmitted)="onTaskSubmitted($event)"
    (modalClosed)="closeTaskModal()">
  </app-task-modal>
</div>
```

**Layout:**

- .main-layout: display: flex, flex: 1
- .content-area: flex: 1, padding: 24px

- loadBoardData() → GET /api/youth-accounts/{id}/dashboard → Load board overview

### 3.2 KanbanColumnComponent

**Component Type:** Feature
**Purpose:** Individual column container (Todo, Progress, Done) with drag-drop functionality
**Parent:** KanbanBoardComponent
**Children:** TaskCardComponent (multiple), AddTaskButtonComponent

**Dependencies:**
- @angular/core
- @angular/cdk/drag-drop

**Library Components Used:**
- Card → Column container
- Button → Add task button

**TypeScript Specification:**

**Inputs:**
- column: Column (column configuration object)
- tasks: Task[] (array of tasks for this column)
- allowDrop: boolean (whether dropping is allowed)

**Outputs:**
- taskMoved: EventEmitter<TaskMoveEvent> (payload: {taskId, fromColumn, toColumn})
- taskClicked: EventEmitter<Task> (payload: Task object)
- addTaskClicked: EventEmitter<string> (payload: columnId)

**State:**
- isDragOver: boolean
- isLoading: boolean
- taskCount: number

**Methods:**
- ngOnInit() → Initialize column
- onDrop(event: CdkDragDrop<Task[]>) → Handle task drop
- onDragOver() → Handle drag over events
- onDragLeave() → Handle drag leave events
- onTaskClick(task: Task) → Handle task selection
- onAddTask() → Trigger add task for this column

**HTML STRUCTURE:**

```html
<div class="kanban-column" 
     [class.drag-over]="isDragOver"
     [attr.aria-label]="column.title + ' column'">
  <div class="column-header">
    <h3>{{ column.title }}</h3>
    <app-badge [count]="taskCount" variant="secondary"></app-badge>
  </div>
  
  <div class="column-body" 
       cdkDropList
       [cdkDropListData]="tasks"
       [cdkDropListDisabled]="!allowDrop"
       (cdkDropListDropped)="onDrop($event)"
       (cdkDropListEntered)="onDragOver()"
       (cdkDropListExited)="onDragLeave()">
    
    <app-task-card 
      *ngFor="let task of tasks; trackBy: trackByTaskId"
      [task]="task"
      [draggable]="true"
      cdkDrag
      (click)="onTaskClick(task)"
      (cdkDragStarted)="onDragStart(task)"
      (cdkDragEnded)="onDragEnd()">
    </app-task-card>
    
    <div *ngIf="tasks.length === 0" class="empty-state">
      <p>No tasks in {{ column.title.toLowerCase() }}</p>
    </div>
  </div>
  
  <div class="column-footer">
    <app-button 
      variant="ghost" 
      (click)="onAddTask()"
      [attr.aria-label]="'Add task to ' + column.title">
      + Add Task
    </app-button>
  </div>
</div>
```

**CSS SPECIFICATION:**

**Layout:**
- .kanban-column: display: flex, flex-direction: column, min-height: 500px
- .column-body: flex: 1, overflow-y: auto

**Spacing:**
- Column padding: 16px
- Task spacing: 12px between cards

**Component Styles:**
- .column-header: display: flex, justify-content: space-between, align-items: center
- .drag-over: background-color: rgba(0, 123, 255, 0.1), border: 2px dashed #007bff
- .empty-state: text-align: center, color: #6c757d, padding: 40px 20px

**Responsive:**
- Mobile: reduce padding to 12px, smaller min-height
