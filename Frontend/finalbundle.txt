# FINAL UI PACKAGE BUNDLE

## 1. JIRA REQUIREMENT SUMMARY

**Story Description:**
AS a parent or guardian I WANT to allocate and manage funds for my child's youth account SO THAT I can teach financial responsibility while maintaining visibility and control over spending

**Acceptance Criteria:**
- Scenario 1 – Access youth account dashboard: Given the parent is logged into the banking web application When the parent navigates to the youth account section Then the system should display the youth account dashboard with current balance and recent activity
- Scenario 2 – Allocate funds to youth account: Given the parent is on the youth account dashboard When the parent selects "Add Funds" Then the system should allow the parent to transfer funds from their primary account to the youth account
- Scenario 3 – Set spending limit: Given the parent is viewing the youth account management section When the parent configures a weekly spending limit Then the system should save and display the limit on the dashboard
- Scenario 4 – View youth spending activity: Given the parent is viewing the youth account dashboard When the parent selects the activity section Then the system should display recent transactions made from the youth account
- Scenario 5 – Handle insufficient balance during fund allocation: Given the parent attempts to transfer funds When the selected parent account does not have sufficient balance Then the system should display an error message indicating insufficient funds

**UI Tasks:**
- Design youth account dashboard layout
- Design fund transfer interaction for youth account
- Design spending limit configuration interface
- Design youth account activity view

### Component Responsibilities:

**YouthAccountDashboard**
- Purpose: Main container component for youth account management
- Props: youthAccountId: string, parentAccountId: string
- State: dashboardData: DashboardData, loading: boolean, error: string
- Events: onRefresh: void, onError: (error: string) => void
- API Binding: GET /api/youth-accounts/{youthAccountId}/dashboard

**AccountOverviewCard**
- Purpose: Display current balance and account summary
- Props: balance: number, currency: string, spendingLimit: number, accountId: string
- State: refreshing: boolean
- Events: onRefresh: void
- API Binding: GET /api/youth-accounts/{youthAccountId}/dashboard

**FundTransferModal**
- Purpose: Handle fund transfer from parent to youth account
- Props: isOpen: boolean, youthAccountId: string, parentAccounts: Account[]
- State: transferAmount: number, sourceAccountId: string, description: string, processing: boolean, validationErrors: ValidationError[]
- Events: onTransfer: (transferData: FundTransferRequest) => void, onClose: void, onSuccess: (result: FundTransferResponse) => void
- API Binding: POST /api/youth-accounts/{youthAccountId}/fund-transfer

**SpendingLimitConfigModal**
- Purpose: Configure weekly spending limits
- Props: isOpen: boolean, youthAccountId: string, currentLimit: number
- State: weeklyLimit: number, effectiveDate: string, processing: boolean
- Events: onSave: (limitData: SpendingLimitRequest) => void, onClose: void
- API Binding: PUT /api/youth-accounts/{youthAccountId}/spending-limit

**ActivityTimeline**
- Purpose: Display transaction history and activity
- Props: youthAccountId: string, pageSize: number
- State: transactions: Transaction[], pagination: PaginationInfo, loading: boolean, filters: TransactionFilters
- Events: onLoadMore: void, onFilter: (filters: TransactionFilters) => void
- API Binding: GET /api/youth-accounts/{youthAccountId}/transactions

**TransactionCard**
- Purpose: Individual transaction display component
- Props: transaction: TransactionDetail, showCategory: boolean
- State: expanded: boolean
- Events: onExpand: void

### Data Flow:
**Parent → Child Mapping:**
- YouthAccountDashboard → AccountOverviewCard (dashboardData.balance, dashboardData.spendingLimit)
- YouthAccountDashboard → FundTransferModal (youthAccountId, parentAccounts)
- YouthAccountDashboard → SpendingLimitConfigModal (youthAccountId, currentLimit)
- YouthAccountDashboard → ActivityTimeline (youthAccountId, pageSize)
- ActivityTimeline → TransactionCard (transaction, showCategory)

```typescript
import { Component, OnInit, Input, Output, EventEmitter } from '@angular/core';
import { YouthAccountService } from '../../../services/YouthAccountService';
import { DashboardData, Account, ValidationError } from '../../../types/api.types';

@Component({
  selector: 'app-youth-account-dashboard',
  templateUrl: './YouthAccountDashboard.component.html',
  styleUrls: ['./YouthAccountDashboard.component.scss']
})
export class YouthAccountDashboardComponent implements OnInit {
  @Input() youthAccountId: string = '';
  @Input() parentAccountId: string = '';
  @Output() onRefresh = new EventEmitter<void>();
  @Output() onError = new EventEmitter<string>();

  // State Management
  dashboardData: DashboardData | null = null;
  loading: boolean = false;
  error: string = '';
  showFundTransferModal: boolean = false;
  showSpendingLimitModal: boolean = false;
  parentAccounts: Account[] = [];

  constructor(private youthAccountService: YouthAccountService) {}

  ngOnInit(): void {
    this.loadDashboardData();
    this.loadParentAccounts();
  }

  async loadDashboardData(): Promise<void> {
    this.loading = true;
    this.error = '';
    
    try {
      this.dashboardData = await this.youthAccountService.getDashboardData(this.youthAccountId);
    } catch (error) {
      this.error = 'Failed to load dashboard data';
      this.onError.emit(this.error);
    } finally {
      this.loading = false;
    }
  }

  async loadParentAccounts(): Promise<void> {
    try {
      this.parentAccounts = await this.youthAccountService.getParentAccounts(this.parentAccountId);
    } catch (error) {
      console.error('Failed to load parent accounts:', error);
    }
  }

  openFundTransferModal(): void {
    this.showFundTransferModal = true;
  }

  closeFundTransferModal(): void {
    this.showFundTransferModal = false;
  }

  openSpendingLimitModal(): void {
    this.showSpendingLimitModal = true;
  }

  closeSpendingLimitModal(): void {
    this.showSpendingLimitModal = false;
  }

  onFundTransferSuccess(): void {
    this.closeFundTransferModal();
    this.loadDashboardData(); // Refresh dashboard data
  }

  onSpendingLimitSuccess(): void {
    this.closeSpendingLimitModal();
    this.loadDashboardData(); // Refresh dashboard data
  }

  refreshDashboard(): void {
    this.loadDashboardData();
    this.onRefresh.emit();
  }
}
```

#### 2. AccountOverviewCard Component

**TypeScript Implementation:**
```typescript
import { Component, Input, Output, EventEmitter, OnInit } from '@angular/core';
import { YouthAccountService } from '../../../services/YouthAccountService';

@Component({
  selector: 'app-account-overview-card',
  templateUrl: './AccountOverviewCard.component.html',
  styleUrls: ['./AccountOverviewCard.component.scss']
})
export class AccountOverviewCardComponent implements OnInit {
  @Input() balance: number = 0;
  @Input() currency: string = 'USD';
  @Input() spendingLimit: number = 0;
  @Input() accountId: string = '';
  @Output() onRefresh = new EventEmitter<void>();

  // State Management
  refreshing: boolean = false;
  spentThisWeek: number = 0;
  remainingLimit: number = 0;
  limitPercentage: number = 0;

  constructor(private youthAccountService: YouthAccountService) {}

  ngOnInit(): void {
    this.calculateSpendingMetrics();
  }

  ngOnChanges(): void {
    this.calculateSpendingMetrics();
  }

  async refreshBalance(): Promise<void> {
    this.refreshing = true;
    
    try {
      await new Promise(resolve => setTimeout(resolve, 1000)); // Simulate API call
      this.onRefresh.emit();
    } catch (error) {
      console.error('Failed to refresh balance:', error);
    } finally {
      this.refreshing = false;
    }
  }

  private calculateSpendingMetrics(): void {
    // Calculate spending metrics based on current data
    this.spentThisWeek = Math.max(0, this.spendingLimit - this.balance);
    this.remainingLimit = Math.max(0, this.spendingLimit - this.spentThisWeek);
    this.limitPercentage = this.spendingLimit > 0 ? (this.spentThisWeek / this.spendingLimit) * 100 : 0;
  }

  formatCurrency(amount: number): string {
    return new Intl.NumberFormat('en-US', {
      style: 'currency',
      currency: this.currency
    }).format(amount);
  }

  getLimitStatusClass(): string {
    if (this.limitPercentage >= 90) return 'danger';
    if (this.limitPercentage >= 70) return 'warning';
    return 'success';
  }
}
```

```typescript
import { Component, Input, Output, EventEmitter, OnInit } from '@angular/core';
import { FormBuilder, FormGroup, Validators } from '@angular/forms';
import { FundTransferService } from '../../../services/FundTransferService';
import { Account, FundTransferRequest, FundTransferResponse, ValidationError } from '../../../types/api.types';

@Component({
  selector: 'app-fund-transfer-modal',
  templateUrl: './FundTransferModal.component.html',
  styleUrls: ['./FundTransferModal.component.scss']
})
export class FundTransferModalComponent implements OnInit {
  @Input() isOpen: boolean = false;
  @Input() youthAccountId: string = '';
  @Input() parentAccounts: Account[] = [];
  @Output() onClose = new EventEmitter<void>();
  @Output() onSuccess = new EventEmitter<FundTransferResponse>();

  // Form Management
  transferForm: FormGroup;
  processing: boolean = false;
  validationErrors: ValidationError[] = [];
  transferSuccess: boolean = false;
  transferResult: FundTransferResponse | null = null;

  constructor(
    private formBuilder: FormBuilder,
    private fundTransferService: FundTransferService
  ) {
    this.transferForm = this.createTransferForm();
  }

  ngOnInit(): void {
    if (this.isOpen) {
      this.resetForm();
    }
  }

  ngOnChanges(): void {
    if (this.isOpen && !this.transferForm.get('sourceAccountId')?.value && this.parentAccounts.length > 0) {
      this.transferForm.patchValue({
        sourceAccountId: this.parentAccounts[0].id
      });
    }
  }

  private createTransferForm(): FormGroup {
    return this.formBuilder.group({
      sourceAccountId: ['', [Validators.required]],
      transferAmount: [0, [Validators.required, Validators.min(0.01), Validators.max(10000)]],
      description: ['', [Validators.maxLength(100)]],
      currency: ['USD']
    });
  }

  resetForm(): void {
    this.transferForm.reset({
      sourceAccountId: this.parentAccounts.length > 0 ? this.parentAccounts[0].id : '',
      transferAmount: 0,
      description: '',
      currency: 'USD'
    });
    this.validationErrors = [];
    this.transferSuccess = false;
    this.transferResult = null;
    this.processing = false;
  }

  async submitTransfer(): Promise<void> {
    if (this.transferForm.invalid) {
      this.markFormGroupTouched();
      return;
    }

    this.processing = true;
    this.validationErrors = [];

    try {
      const transferData: FundTransferRequest = {
        sourceAccountId: this.transferForm.value.sourceAccountId,
        targetAccountId: this.youthAccountId,
        amount: this.transferForm.value.transferAmount,
        currency: this.transferForm.value.currency,
        description: this.transferForm.value.description || 'Fund transfer to youth account',
        transferType: 'PARENT_TO_YOUTH'
      };

      this.transferResult = await this.fundTransferService.transferFunds(transferData);
      this.transferSuccess = true;
      
      // Auto-close after success
      setTimeout(() => {
        this.onSuccess.emit(this.transferResult!);
        this.closeModal();
      }, 2000);

    } catch (error: any) {
      this.handleTransferError(error);
    } finally {
      this.processing = false;
    }
  }

  private handleTransferError(error: any): void {
    if (error.status === 422 && error.error?.validationErrors) {
      this.validationErrors = error.error.validationErrors;
    } else if (error.status === 400 && error.error?.message?.includes('insufficient')) {
      this.validationErrors = [{
        field: 'transferAmount',
        message: 'Insufficient funds in the selected account',
        code: 'INSUFFICIENT_FUNDS'
      }];
    } else {
      this.validationErrors = [{
        field: 'general',
        message: 'Transfer failed. Please try again.',
        code: 'TRANSFER_FAILED'
      }];
    }
  }

  private markFormGroupTouched(): void {
    Object.keys(this.transferForm.controls).forEach(key => {
      this.transferForm.get(key)?.markAsTouched();
    });
  }

  closeModal(): void {
    this.resetForm();
    this.onClose.emit();
  }

  getSelectedAccount(): Account | undefined {
    const selectedId = this.transferForm.get('sourceAccountId')?.value;
    return this.parentAccounts.find(account => account.id === selectedId);
  }

  getFieldError(fieldName: string): string | null {
    const control = this.transferForm.get(fieldName);
    const validationError = this.validationErrors.find(error => error.field === fieldName);
    
    if (validationError) {
      return validationError.message;
    }
    
    if (control?.invalid && control?.touched) {
      if (control.errors?.['required']) {
        return `${fieldName} is required`;
      }
      if (control.errors?.['min']) {
        return `Amount must be greater than $0.01`;
      }
      if (control.errors?.['max']) {
        return `Amount cannot exceed $10,000`;
      }
      if (control.errors?.['maxlength']) {
        return `Description cannot exceed 100 characters`;
      }
    }
    
    return null;
  }

  formatCurrency(amount: number): string {
    return new Intl.NumberFormat('en-US', {
      style: 'currency',
      currency: 'USD'
    }).format(amount);
  }
}
```

```typescript
      notifyAt90Percent: [true]
    });
  }

  resetForm(): void {
    this.limitForm.reset({
      weeklyLimit: this.currentLimit || 0,
      effectiveDate: new Date().toISOString().split('T')[0],
      notifyWhenReached: true,
      notifyAt90Percent: true
    });
    this.saveSuccess = false;
    this.errorMessage = '';
    this.processing = false;
  }

  async saveSpendingLimit(): Promise<void> {
    if (this.limitForm.invalid) {
      this.markFormGroupTouched();
      return;
    }

    this.processing = true;
    this.errorMessage = '';

    try {
      const limitData: SpendingLimitRequest = {
        weeklyLimit: this.limitForm.value.weeklyLimit,
        effectiveDate: this.limitForm.value.effectiveDate,
        notificationSettings: {
          notifyWhenReached: this.limitForm.value.notifyWhenReached,
          notifyAt90Percent: this.limitForm.value.notifyAt90Percent
        }
      };

      const result = await this.youthAccountService.updateSpendingLimit(this.youthAccountId, limitData);
      this.saveSuccess = true;
      
      // Auto-close after success
      setTimeout(() => {
        this.onSuccess.emit(result);
        this.closeModal();
      }, 2000);

    } catch (error: any) {
      this.errorMessage = error.error?.message || 'Failed to update spending limit. Please try again.';
    } finally {
      this.processing = false;
    }
  }

  removeSpendingLimit(): void {
    this.limitForm.patchValue({ weeklyLimit: 0 });
  }

  setQuickLimit(amount: number): void {
    this.limitForm.patchValue({ weeklyLimit: amount });
  }

  private markFormGroupTouched(): void {
    Object.keys(this.limitForm.controls).forEach(key => {
      this.limitForm.get(key)?.markAsTouched();
    });
  }

  closeModal(): void {
    this.resetForm();
    this.onClose.emit();
  }

  getFieldError(fieldName: string): string | null {
    const control = this.limitForm.get(fieldName);
    
    if (control?.invalid && control?.touched) {
      if (control.errors?.['required']) {
        return `${fieldName} is required`;
      }
      if (control.errors?.['min']) {
        return `Amount must be $0 or greater`;
      }
      if (control.errors?.['max']) {
        return `Amount cannot exceed $10,000`;
      }
    }
    
    return null;
  }

  formatCurrency(amount: number): string {
    return new Intl.NumberFormat('en-US', {
      style: 'currency',
      currency: 'USD'
    }).format(amount);
  }

  isLimitChanged(): boolean {
    return this.limitForm.get('weeklyLimit')?.value !== this.currentLimit;
  }

  getLimitStatus(): { type: string; message: string } {
    const newLimit = this.limitForm.get('weeklyLimit')?.value || 0;
    
    if (newLimit === 0) {
      return {
        type: 'info',
        message: 'No spending limit will be set'
      };
    }
    
    if (newLimit > this.currentLimit) {
      return {
        type: 'success',
        message: `Spending limit will be increased to ${this.formatCurrency(newLimit)}`
      };
    }
    
    if (newLimit < this.currentLimit) {
      return {
        type: 'warning',
        message: `Spending limit will be reduced to ${this.formatCurrency(newLimit)}`
      };
    }
    
    return {
      type: 'info',
      message: `Spending limit remains ${this.formatCurrency(newLimit)}`
    };
  }
}
```

**TypeScript Implementation:**
```typescript
import { Component, Input, OnInit, OnDestroy } from '@angular/core';
import { TransactionService } from '../../../services/TransactionService';
import { Transaction, TransactionFilters, PaginationInfo } from '../../../types/api.types';
import { Subject } from 'rxjs';
import { debounceTime, distinctUntilChanged, takeUntil } from 'rxjs/operators';

@Component({
  selector: 'app-activity-timeline',
  templateUrl: './ActivityTimeline.component.html',
  styleUrls: ['./ActivityTimeline.component.scss']
})
export class ActivityTimelineComponent implements OnInit, OnDestroy {
  @Input() youthAccountId: string = '';
  @Input() pageSize: number = 10;

  // State Management
  transactions: Transaction[] = [];
  pagination: PaginationInfo = {
    currentPage: 1,
    totalPages: 1,
    totalItems: 0,
    hasNextPage: false,
    hasPreviousPage: false
  };
  loading: boolean = false;
  loadingMore: boolean = false;
  error: string = '';
  
  // Filtering
  filters: TransactionFilters = {
    dateFrom: '',
    dateTo: '',
    category: '',
    minAmount: null,
    maxAmount: null,
    searchTerm: ''
  };
  
  showFilters: boolean = false;
  searchSubject = new Subject<string>();
  private destroy$ = new Subject<void>();
  
  // Filter options
  categoryOptions = [
    { value: '', label: 'All Categories' },
    { value: 'food', label: 'Food & Dining' },
    { value: 'entertainment', label: 'Entertainment' },
    { value: 'shopping', label: 'Shopping' },
    { value: 'transportation', label: 'Transportation' },
    { value: 'education', label: 'Education' },
    { value: 'other', label: 'Other' }
  ];

  constructor(private transactionService: TransactionService) {}

  ngOnInit(): void {
    this.setupSearchDebounce();
    this.loadTransactions();
  }

  ngOnDestroy(): void {
    this.destroy$.next();
    this.destroy$.complete();
  }

  private setupSearchDebounce(): void {
    this.searchSubject.pipe(
      debounceTime(300),
      distinctUntilChanged(),
      takeUntil(this.destroy$)
    ).subscribe(searchTerm => {
      this.filters.searchTerm = searchTerm;
      this.loadTransactions(true);
    });
  }

  async loadTransactions(reset: boolean = false): Promise<void> {
    if (reset) {
      this.pagination.currentPage = 1;
      this.transactions = [];
    }
    
    this.loading = reset;
    this.loadingMore = !reset;
    this.error = '';

    try {
      const response = await this.transactionService.getTransactions(
        this.youthAccountId,
        {
          page: this.pagination.currentPage,
          pageSize: this.pageSize,
          ...this.filters
        }
      );

      if (reset) {
        this.transactions = response.transactions;
      } else {
        this.transactions = [...this.transactions, ...response.transactions];
      }
      
      this.pagination = response.pagination;

    } catch (error: any) {
      this.error = 'Failed to load transactions. Please try again.';
      console.error('Transaction loading error:', error);
    } finally {
      this.loading = false;
      this.loadingMore = false;
    }
  }

  async loadMoreTransactions(): Promise<void> {
    if (this.pagination.hasNextPage && !this.loadingMore) {
      this.pagination.currentPage++;
      await this.loadTransactions();
    }
  }

  onSearchChange(searchTerm: string): void {
    this.searchSubject.next(searchTerm);
  }

  onFilterChange(): void {
    this.loadTransactions(true);
  }

  clearFilters(): void {
    this.filters = {
      dateFrom: '',
      dateTo: '',
      category: '',
      minAmount: null,
      maxAmount: null,
      searchTerm: ''
    };
    this.loadTransactions(true);
  }

  toggleFilters(): void {
    this.showFilters = !this.showFilters;
  }

  refreshTransactions(): void {
    this.loadTransactions(true);
  }

  hasActiveFilters(): boolean {
    return !!(this.filters.dateFrom || this.filters.dateTo || this.filters.category || 
             this.filters.minAmount || this.filters.maxAmount || this.filters.searchTerm);
  }

  getTransactionsByDate(): { [date: string]: Transaction[] } {
    const grouped: { [date: string]: Transaction[] } = {};
    
    this.transactions.forEach(transaction => {
      const date = new Date(transaction.date).toDateString();
      if (!grouped[date]) {
        grouped[date] = [];
      }
      grouped[date].push(transaction);
    });
    
    return grouped;
  }

  formatDate(dateString: string): string {
    const date = new Date(dateString);
    const today = new Date();
    const yesterday = new Date(today);
    yesterday.setDate(yesterday.getDate() - 1);
    
    if (date.toDateString() === today.toDateString()) {
      return 'Today';
    } else if (date.toDateString() === yesterday.toDateString()) {
      return 'Yesterday';
    } else {
      return date.toLocaleDateString('en-US', { 
        weekday: 'long', 
        year: 'numeric', 
        month: 'long', 
        day: 'numeric' 
      });
    }
  }

  getEmptyStateMessage(): string {
    if (this.hasActiveFilters()) {
      return 'No transactions found matching your filters.';
    }
    return 'No transactions yet. Start spending to see activity here!';
  }
}
```

**TypeScript Implementation:**
```typescript
import { Component, Input } from '@angular/core';
import { TransactionDetail } from '../../../types/api.types';

@Component({
  selector: 'app-transaction-card',
  templateUrl: './TransactionCard.component.html',
  styleUrls: ['./TransactionCard.component.scss']
})
export class TransactionCardComponent {
  @Input() transaction!: TransactionDetail;
  @Input() showCategory: boolean = true;

  // State Management
  expanded: boolean = false;

  toggleExpanded(): void {
    this.expanded = !this.expanded;
  }

  formatCurrency(amount: number): string {
    return new Intl.NumberFormat('en-US', {
      style: 'currency',
      currency: 'USD'
    }).format(Math.abs(amount));
  }

  formatTime(dateString: string): string {
    return new Date(dateString).toLocaleTimeString('en-US', {
      hour: '2-digit',
      minute: '2-digit'
    });
  }

  getCategoryIcon(category: string): string {
    const iconMap: { [key: string]: string } = {
      'food': 'icon-utensils',
      'entertainment': 'icon-film',
      'shopping': 'icon-shopping-bag',
      'transportation': 'icon-car',
      'education': 'icon-book',
      'other': 'icon-more-horizontal'
    };
    return iconMap[category] || 'icon-more-horizontal';
  }

  getCategoryColor(category: string): string {
    const colorMap: { [key: string]: string } = {
      'food': '#f59e0b',
      'entertainment': '#8b5cf6',
      'shopping': '#ec4899',
      'transportation': '#06b6d4',
      'education': '#10b981',
      'other': '#6b7280'
    };
    return colorMap[category] || '#6b7280';
  }

  getTransactionTypeClass(): string {
    return this.transaction.amount < 0 ? 'debit' : 'credit';
  }

  getStatusClass(): string {
    switch (this.transaction.status) {
      case 'completed': return 'status-completed';
      case 'pending': return 'status-pending';
      case 'failed': return 'status-failed';
      default: return 'status-unknown';
    }
  }

  getStatusIcon(): string {
    switch (this.transaction.status) {
      case 'completed': return 'icon-check-circle';
      case 'pending': return 'icon-clock';
      case 'failed': return 'icon-x-circle';
      default: return 'icon-help-circle';
    }
  }

  hasAdditionalDetails(): boolean {
    return !!(this.transaction.description || 
             this.transaction.location || 
             this.transaction.reference ||
             this.transaction.notes);
  }
}
```

### API INTEGRATION MAPPING

**Component → API Endpoint Mapping:**

1. **YouthAccountDashboard**
   - GET /api/youth-accounts/{youthAccountId}/dashboard
   - Fetches: balance, spendingLimit, recentTransactions, accountInfo

2. **AccountOverviewCard**
   - Uses dashboard data from parent component
   - Refresh triggers parent dashboard reload

3. **FundTransferModal**
   - POST /api/youth-accounts/{youthAccountId}/fund-transfer
   - Payload: sourceAccountId, targetAccountId, amount, currency, description, transferType
   - Response: transactionId, amount, status, timestamp

4. **SpendingLimitConfigModal**
   - PUT /api/youth-accounts/{youthAccountId}/spending-limit
   - Payload: weeklyLimit, effectiveDate, notificationSettings
   - Response: updatedLimit, effectiveDate, status

5. **ActivityTimeline**
   - GET /api/youth-accounts/{youthAccountId}/transactions
   - Query params: page, pageSize, dateFrom, dateTo, category, minAmount, maxAmount, searchTerm
   - Response: transactions[], pagination info

6. **TransactionCard**
   - Display component only, no direct API calls
   - Receives transaction data from ActivityTimeline parent
