### Component Responsibilities:

**KanbanBoardComponent:**
- Main container for the three-column Kanban board
- Props: columns: KanbanColumn[], loading: boolean, error: string | null
- State: columns, selectedTask, isLoading, errorMessage
- Events: taskCreated, taskUpdated, taskMoved
- API Binding: GET /api/youth-accounts/{youthAccountId}/dashboard, GET /api/youth-accounts/{youthAccountId}/transactions

**KanbanColumnComponent:**
- Individual column container with header and task list
- Props: column: KanbanColumn, tasks: Task[], allowDrop: boolean
- State: isExpanded, dragOver
- Events: taskDropped, addTaskClicked, columnOptionsClicked

**TaskCardComponent:**
- Individual task card display with all task information
- Props: task: Task, draggable: boolean, selectable: boolean
- State: isDragging, isSelected, isHovered
- Events: taskClicked, taskDragStart, taskDragEnd

**HeaderComponent:**
- Top navigation header with search and user actions
- Props: user: User, showSearch: boolean
- State: searchQuery, notificationCount
- Events: searchChanged, notificationClicked, settingsClicked, profileClicked

**SidebarNavigationComponent:**
- Left sidebar navigation with menu items and quick actions
- Props: currentPage: string, navigationItems: NavigationItem[]
- State: isCollapsed, activeItem
- Events: navigationChanged, quickActionClicked

**TaskModalComponent:**
- Modal dialog for creating and editing tasks
- Props: task: Task | null, assignees: User[], columns: string[]
- State: formData, isSubmitting, validationErrors
- Events: taskSaved, modalClosed
- API Binding: POST /api/youth-accounts/{youthAccountId}/fund-transfer, PUT /api/youth-accounts/{youthAccountId}/spending-limit

### Data Flow:
**Parent → Child Mapping:**
- AppComponent → SidebarNavigationComponent (navigationItems, currentPage)
- AppComponent → MainContentComponent (user, currentPage)
- MainContentComponent → HeaderComponent (user, showSearch)
- MainContentComponent → ContentAreaComponent (currentPage)
- ContentAreaComponent → KanbanBoardComponent (columns, tasks, loading, error)
- KanbanBoardComponent → KanbanColumnComponent (column, tasks, allowDrop)
- KanbanColumnComponent → TaskCardComponent (task, draggable, selectable)
- KanbanBoardComponent → TaskModalComponent (task, assignees, columns)

### API Summary:
**Available APIs:**
- GET /api/youth-accounts/{youthAccountId}/dashboard → Fetch dashboard data with account balance and recent transactions
- POST /api/youth-accounts/{youthAccountId}/fund-transfer → Transfer funds (mapped to task creation)
- PUT /api/youth-accounts/{youthAccountId}/spending-limit → Configure spending limit (mapped to task updates)
- GET /api/youth-accounts/{youthAccountId}/transactions → Get transaction history (mapped to task history)
