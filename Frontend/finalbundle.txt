### 3.1 KanbanBoardComponent

**Component Type:** Feature
**Mapped From A1:** Main container for the kanban board with three columns
**Purpose:** Orchestrates the entire kanban board functionality including task management, drag-and-drop, and filtering

**Parent:** App Component
**Children:** KanbanColumnComponent (3 instances), TaskModalComponent, BoardControlsComponent

**Dependencies:**
- KanbanService
- TaskService
- Angular CDK Drag Drop

**Library Components Used:**
- Button → Add Task actions
- Modal → Task creation dialog
- Badge → Status indicators

**TypeScript Specification:**

**Inputs:**
- @Input() initialTasks: Task[]
- @Input() columns: KanbanColumn[]
- @Input() loading: boolean

**Outputs:**
- @Output() taskCreated: EventEmitter<Task>
- @Output() taskUpdated: EventEmitter<Task>
- @Output() filterChanged: EventEmitter<FilterState>

**State:**
- selectedFilters: FilterState
- draggedTask: Task | null
- isModalOpen: boolean
- errorState: string | null

**Methods:**
- ngOnInit() → Initialize component and load data
- onTaskMove(taskId: string, targetColumn: string) → Handle drag-drop
- onTaskCreate(task: Task) → Create new task
- onFilterChange(filters: FilterState) → Apply filters
- handleError(error: any) → Error handling

**HTML STRUCTURE:**
```html
<div class="kanban-board-container" [attr.aria-label]="'Kanban Board'">
  <div class="board-header">
    <app-page-header [title]="'Kanban Board'" [breadcrumbs]="breadcrumbs"></app-page-header>
    <app-board-controls (filterChange)="onFilterChange($event)"></app-board-controls>
  </div>
  
  <div class="kanban-grid" cdkDropListGroup>
    <app-kanban-column 
      *ngFor="let column of columns" 
      [column]="column"
      [tasks]="getTasksForColumn(column.id)"
      (taskDrop)="onTaskMove($event.taskId, column.id)"
      (addTask)="openTaskModal(column.id)">
    </app-kanban-column>
  </div>
  
  <app-task-modal 
    [isOpen]="isModalOpen"
    [task]="selectedTask"
    [columns]="columns"
    (submit)="onTaskCreate($event)"
    (close)="closeModal()">
  </app-task-modal>
  
  <div *ngIf="errorState" class="error-state" role="alert">
    <p>{{ errorState }}</p>
    <app-button (click)="retryLoad()">Retry</app-button>
  </div>
</div>
```

**CSS SPECIFICATION:**

**Layout:**
- display: flex; flex-direction: column; height: 100vh
- .kanban-grid: display: grid; grid-template-columns: repeat(3, 1fr); gap: 1rem

**Spacing:**
- padding: 1rem
- gap: 1rem between columns

**Component Styles:**
- .kanban-board-container: background: #f5f5f5
- .error-state: background: #fee; border: 1px solid #fcc; padding: 1rem

**Responsive:**
- Desktop (>1024px): 3 columns side by side
- Tablet (768px-1024px): 2 columns with scroll
- Mobile (<768px): Single column with tabs

**API INTEGRATION:**
- ngOnInit() → GET /api/tasks → Load initial tasks
- onTaskMove() → PUT /api/tasks/:id → Update task status
- onTaskCreate() → POST /api/tasks → Create new task

**ERROR HANDLING:**
- API failure → show error state template with retry button
- Validation error → show inline validation messages
- Network error → show offline indicator

**INTERACTION FLOW:**
User Action → UI → API
- Load board → ngOnInit() → GET /api/tasks → update state
- Drag task → onDragEnd() → PUT /api/tasks/:id → refresh UI
- Add task → click button → open modal → submit → POST /api/tasks
