### Component Responsibilities:

**KanbanBoardComponent:**
- Purpose: Main container for three-column Kanban layout
- Props: columns: KanbanColumn[], loading: boolean, error: string
- State: draggedTask: Task | null, activeColumn: string
- Events: taskMoved, taskCreated, taskUpdated, taskDeleted
- API Binding: GET /api/youth-accounts/{id}/dashboard for task data

**KanbanColumnComponent:**
- Purpose: Individual column container (To Do, In Progress, Done)
- Props: column: KanbanColumn, tasks: Task[], allowDrop: boolean
- State: isDropTarget: boolean, taskCount: number
- Events: taskDropped, addTaskClicked
- API Binding: None (receives data from parent)

**TaskCardComponent:**
- Purpose: Individual task display and interaction
- Props: task: Task, draggable: boolean, editable: boolean
- State: isDragging: boolean, isSelected: boolean
- Events: taskClicked, taskDragStart, taskDragEnd
- API Binding: PUT /api/youth-accounts/{id}/spending-limit for task updates

**TaskModalComponent:**
- Purpose: Create/edit task dialog
- Props: isOpen: boolean, task: Task | null, mode: 'create' | 'edit'
- State: formData: TaskFormData, isSubmitting: boolean, validationErrors: ValidationError[]
- Events: taskSaved, modalClosed
- API Binding: POST /api/youth-accounts/{id}/fund-transfer for task creation

**HeaderComponent:**
- Purpose: Top navigation and user controls
- Props: user: User, breadcrumbs: Breadcrumb[]
- State: searchQuery: string, isUserMenuOpen: boolean
- Events: searchSubmitted, themeToggled, userMenuToggled
- API Binding: GET /api/youth-accounts/{id}/transactions for search

**SidebarComponent:**
- Purpose: Left navigation menu
- Props: navigationItems: NavItem[], activeRoute: string
- State: isCollapsed: boolean, isMobileOpen: boolean
- Events: navigationClicked, sidebarToggled
- API Binding: None (static navigation)

### Data Flow:
Parent → Child Mapping:
```
AppComponent → HeaderComponent (user, breadcrumbs)
AppComponent → SidebarComponent (navigationItems, activeRoute)
AppComponent → KanbanBoardComponent (columns, tasks, loading)
KanbanBoardComponent → KanbanColumnComponent (column, tasks, allowDrop)
KanbanColumnComponent → TaskCardComponent (task, draggable, editable)
AppComponent → TaskModalComponent (isOpen, task, mode)
```
