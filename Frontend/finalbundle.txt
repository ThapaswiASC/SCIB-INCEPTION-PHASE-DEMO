### 3.1 KanbanBoardComponent

**Component Type:** Feature
**Mapped From A1:** Main container for three-column kanban layout
**Purpose:** Primary container managing kanban board state and three-column layout with drag/drop functionality
**Parent:** ContentAreaComponent
**Children:**
- KanbanColumnComponent (To Do)
- KanbanColumnComponent (In Progress)
- KanbanColumnComponent (Done)

**Dependencies:**
- KanbanService
- TaskService
- Angular CDK Drag Drop

**Library Components Used:**
- aava-button → Action buttons
- aava-dialog → Error dialogs

**TypeScript Specification**

**Inputs:**
- loading: boolean (loading state indicator)
- error: string | null (error message display)

**Outputs:**
- taskSelected: EventEmitter<Task> (task selection events)
- taskMoved: EventEmitter<TaskMoveEvent> (task drag/drop events)
- columnAction: EventEmitter<ColumnActionEvent> (column-level actions)

**State:**
- columns: KanbanColumn[] (column data with tasks)
- isLoading: boolean (loading state)
- errorMessage: string | null (error handling)

**Methods:**
- ngOnInit() → Initialize component and load data
- loadKanbanData() → Fetch initial kanban data
- onTaskMoved(event: CdkDragDrop<Task[]>) → Handle task drag/drop
- onTaskSelected(task: Task) → Handle task selection
- handleError(error: any) → Error handling

**HTML STRUCTURE (PSEUDO CODE ONLY)**

```html
<div class="kanban-board-container" role="region" aria-label="Kanban Board">
  <div class="kanban-header">
    <h2>Task Board</h2>
    <aava-button variant="primary" (click)="openCreateTaskModal()">Create Task</aava-button>
  </div>
  
  <div class="kanban-grid" cdkDropListGroup>
    <div class="kanban-column" *ngFor="let column of columns">
      <app-kanban-column 
        [column]="column"
        [allowDrop]="true"
        (taskDropped)="onTaskMoved($event)"
        (taskSelected)="onTaskSelected($event)">
      </app-kanban-column>
    </div>
  </div>
  
  <div class="error-container" *ngIf="errorMessage">
    <aava-dialog [isOpen]="!!errorMessage" (closed)="clearError()">
      <p>{{ errorMessage }}</p>
    </aava-dialog>
  </div>
</div>
```

**CSS SPECIFICATION**

**Layout:**
- .kanban-board-container: flex column layout
- .kanban-grid: display: grid; grid-template-columns: repeat(3, 1fr); gap: 1rem

**Spacing:**
- padding: 1.5rem
- gap: 1rem between columns

**Component Styles:**
- .kanban-header: flex justify-between align-center
- .kanban-column: min-height: 500px
- .error-container: position fixed for modal overlay

**Responsive:**
- Desktop (>1024px): 3 columns side by side
- Tablet (768px-1024px): 2 columns with scroll
- Mobile (<768px): 1 column stacked

**API INTEGRATION**
- loadKanbanData() → GET /api/youth-accounts/{id}/dashboard → Load initial kanban data
- onTaskMoved() → PUT /api/tasks/{id}/status → Update task status on move

**ERROR HANDLING**
- API failure → show error dialog with retry option
- Validation error → inline column error messages
- Network timeout → show connection error with offline mode

**INTERACTION FLOW**
**User Action → UI → API**
- Component init → load columns → GET /api/dashboard → display kanban board
- Drag task → update UI → PUT /api/tasks/{id} → sync status
- Click create → open modal → POST /api/tasks → refresh board

### 3.2 KanbanColumnComponent

**Component Type:** Feature
**Mapped From A1:** Individual column with header and task list
**Purpose:** Manages individual kanban column with drag/drop zone and task list display
**Parent:** KanbanBoardComponent
**Children:**
- TaskCardComponent[] (multiple task cards)
- ColumnHeaderComponent

**Dependencies:**
- Angular CDK Drag Drop
- TaskService

**Library Components Used:**
- aava-button → Column action buttons
- aava-popover → Column options menu

**TypeScript Specification**

**Inputs:**
- column: KanbanColumn (column configuration and data)
- allowDrop: boolean (enable drag/drop functionality)
- readonly: boolean (read-only mode)

**Outputs:**
- taskDropped: EventEmitter<TaskDropEvent> (task drop events)
- addTask: EventEmitter<string> (add task to column)
- columnOptionsClicked: EventEmitter<string> (column menu actions)
- taskSelected: EventEmitter<Task> (task selection from child)

**State:**
- tasks: Task[] (tasks in this column)
- isDropTarget: boolean (drag/drop state)
- showOptions: boolean (column options menu)

**Methods:**
- onTaskDrop(event: CdkDragDrop<Task[]>) → Handle task drop
- onAddTask() → Trigger add task event
- toggleOptions() → Show/hide column options
- onTaskClick(task: Task) → Forward task selection

**HTML STRUCTURE (PSEUDO CODE ONLY)**

```html
<div class="kanban-column" [attr.aria-labelledby]="column.id + '-header'">
  <div class="column-header" [id]="column.id + '-header'">
    <h3>{{ column.title }}</h3>
    <span class="task-count">{{ column.tasks.length }}</span>
    <aava-button variant="ghost" (click)="toggleOptions()" aria-label="Column options">
      <i class="icon-more"></i>
    </aava-button>
    <aava-popover [isOpen]="showOptions" (closed)="showOptions = false">
      <button (click)="onAddTask()">Add Task</button>
      <button (click)="clearColumn()">Clear Column</button>
    </aava-popover>
  </div>
  
  <div class="task-list" 
       cdkDropList 
       [cdkDropListData]="column.tasks"
       (cdkDropListDropped)="onTaskDrop($event)"
       [class.drop-target]="isDropTarget">
    <app-task-card 
      *ngFor="let task of column.tasks; trackBy: trackByTaskId"
      [task]="task"
      [draggable]="allowDrop && !readonly"
      (taskClicked)="onTaskClick(task)"
      cdkDrag>
    </app-task-card>
    
    <div class="empty-state" *ngIf="column.tasks.length === 0">
      <p>No tasks in {{ column.title }}</p>
      <aava-button variant="secondary" (click)="onAddTask()" *ngIf="!readonly">
        Add First Task
      </aava-button>
    </div>
  </div>
</div>
```

**CSS SPECIFICATION**

**Layout:**
- .kanban-column: flex column, min-height: 500px
- .task-list: flex column with gap, flex-grow: 1

**Spacing:**
- padding: 1rem
- gap: 0.75rem between tasks

**Component Styles:**
- .column-header: flex justify-between, border-bottom
- .drop-target: border-dashed, background highlight
- .empty-state: text-center, padding: 2rem

**Responsive:**
- Desktop: full height columns
- Tablet: scrollable columns
- Mobile: stacked layout

**API INTEGRATION**
- No direct API calls (managed by parent KanbanBoardComponent)
- Data flows through @Input properties

**ERROR HANDLING**
- Drop validation → show inline error message
- Task load failure → show column-level error state
- Permission errors → disable drag/drop functionality

**INTERACTION FLOW**
**User Action → UI → API**
- Drop task → validate drop → emit taskDropped → parent handles API
- Click add task → emit addTask → parent opens modal
- Click options → show popover → emit columnOptionsClicked

### 3.3 TaskCardComponent

**Component Type:** Shared
**Mapped From A1:** Individual task display with all task information
**Purpose:** Displays individual task with priority, title, description, tags, assignee, and due date
**Parent:** KanbanColumnComponent
**Children:** None (leaf component)

**Dependencies:**
- DatePipe
- Angular CDK Drag Drop

**Library Components Used:**
- aava-popup → Task details tooltip
- aava-button → Quick action buttons

**TypeScript Specification**

**Inputs:**
- task: Task (task data object)
- draggable: boolean (enable drag functionality)
- selectable: boolean (enable selection)
- compact: boolean (compact display mode)

**Outputs:**
- taskClicked: EventEmitter<Task> (task selection)
- taskDragStart: EventEmitter<Task> (drag start)
- taskDragEnd: EventEmitter<Task> (drag end)
- quickAction: EventEmitter<{action: string, task: Task}> (quick actions)

**State:**
- isDragging: boolean (drag state)
- isSelected: boolean (selection state)
- showTooltip: boolean (tooltip visibility)

**Methods:**
- onClick() → Emit task selection
- onDragStart() → Set drag state and emit event
- onDragEnd() → Clear drag state and emit event
- onQuickAction(action: string) → Emit quick action
- getPriorityClass() → Return CSS class for priority
- formatDueDate() → Format due date display

**HTML STRUCTURE (PSEUDO CODE ONLY)**

```html
<div class="task-card" 
     [class.dragging]="isDragging"
     [class.selected]="isSelected"
     [class.compact]="compact"
     (click)="onClick()"
     (cdkDragStarted)="onDragStart()"
     (cdkDragEnded)="onDragEnd()"
     [attr.aria-label]="'Task: ' + task.title">
  
  <div class="task-header">
    <span class="priority-indicator" [class]="getPriorityClass()"></span>
    <h4 class="task-title">{{ task.title }}</h4>
    <aava-button variant="ghost" size="small" (click)="onQuickAction('edit')" *ngIf="!compact">
      <i class="icon-edit"></i>
    </aava-button>
  </div>
  
  <p class="task-description" *ngIf="task.description && !compact">
    {{ task.description | slice:0:100 }}{{ task.description.length > 100 ? '...' : '' }}
  </p>
  
  <div class="task-tags" *ngIf="task.tags?.length && !compact">
    <span class="tag" *ngFor="let tag of task.tags | slice:0:3">{{ tag }}</span>
    <span class="tag-more" *ngIf="task.tags.length > 3">+{{ task.tags.length - 3 }}</span>
  </div>
  
  <div class="task-footer">
    <div class="assignee" *ngIf="task.assignee">
      <img [src]="task.assignee.avatar" [alt]="task.assignee.name" class="avatar">
      <span class="assignee-name" *ngIf="!compact">{{ task.assignee.name }}</span>
    </div>
    
    <div class="due-date" *ngIf="task.dueDate" [class.overdue]="isOverdue(task.dueDate)">
      <i class="icon-calendar"></i>
      <span>{{ formatDueDate() }}</span>
    </div>
  </div>
  
  <aava-popup [isOpen]="showTooltip" [target]="taskCard" position="top">
    <div class="task-tooltip">
      <h5>{{ task.title }}</h5>
      <p>{{ task.description }}</p>
      <div class="tooltip-meta">
        <span>Priority: {{ task.priority }}</span>
        <span>Status: {{ task.status }}</span>
        <span *ngIf="task.dueDate">Due: {{ task.dueDate | date:'short' }}</span>
      </div>
    </div>
  </aava-popup>
</div>
```
