# FINAL UI PACKAGE BUNDLE

## 1. JIRA REQUIREMENT SUMMARY

**Story Description:**
AS a parent or guardian I WANT to allocate and manage funds for my child's youth account SO THAT I can teach financial responsibility while maintaining visibility and control over spending

**Acceptance Criteria:**
- Scenario 1 – Access youth account dashboard: Given the parent is logged into the banking web application When the parent navigates to the youth account section Then the system should display the youth account dashboard with current balance and recent activity
- Scenario 2 – Allocate funds to youth account: Given the parent is on the youth account dashboard When the parent selects "Add Funds" Then the system should allow the parent to transfer funds from their primary account to the youth account
- Scenario 3 – Set spending limit: Given the parent is viewing the youth account management section When the parent configures a weekly spending limit Then the system should save and display the limit on the dashboard
- Scenario 4 – View youth spending activity: Given the parent is viewing the youth account dashboard When the parent selects the activity section Then the system should display recent transactions made from the youth account
- Scenario 5 – Handle insufficient balance during fund allocation: Given the parent attempts to transfer funds When the selected parent account does not have sufficient balance Then the system should display an error message indicating insufficient funds

**UI Tasks:**
- Design youth account dashboard layout
- Design fund transfer interaction for youth account
- Design spending limit configuration interface
- Design youth account activity view

### Component Responsibilities:

**YouthAccountApp:**
- Purpose: Root application component managing global state and routing
- Props: userId: string, youthAccountId: string
- State: currentView: string, userSession: object, dashboardData: object
- Events: onViewChange, onSessionExpire
- API Binding: Authentication validation

**HeaderComponent:**
- Purpose: Application header with navigation and user info
- Props: userName: string, userAvatar: string, showSearch: boolean
- State: searchQuery: string
- Events: onSearch, onProfileClick
- API Binding: None

**DashboardOverviewComponent:**
- Purpose: Main dashboard showing account summary
- Props: accountBalance: number, spendingLimit: number, currency: string
- State: isLoading: boolean, refreshing: boolean
- Events: onRefresh, onAddFunds, onSetLimit
- API Binding: GET /api/youth-accounts/{youthAccountId}/dashboard

**AccountBalanceCard:**
- Purpose: Display current account balance
- Props: balance: number, currency: string, lastUpdated: string
- State: None
- Events: onRefresh
- API Binding: None (receives data from parent)

**SpendingLimitCard:**
- Purpose: Display and manage spending limits
- Props: currentLimit: number, currency: string, weeklySpent: number
- State: isEditing: boolean
- Events: onEditLimit, onSaveLimit
- API Binding: PUT /api/youth-accounts/{youthAccountId}/spending-limit

**FundTransferComponent:**
- Purpose: Handle fund transfers to youth account
- Props: sourceAccounts: array, youthAccountId: string
- State: transferAmount: number, selectedSource: string, isProcessing: boolean, transferResult: object
- Events: onTransferSubmit, onTransferCancel
- API Binding: POST /api/youth-accounts/{youthAccountId}/fund-transfer

**ActivityTimelineComponent:**
- Purpose: Display transaction history and activity
- Props: youthAccountId: string, pageSize: number
- State: transactions: array, currentPage: number, isLoading: boolean, filters: object
- Events: onPageChange, onFilterChange, onTransactionClick
- API Binding: GET /api/youth-accounts/{youthAccountId}/transactions

**TransactionListComponent:**
- Purpose: Render list of transactions
- Props: transactions: array, currency: string
- State: None
- Events: onTransactionSelect
- API Binding: None (receives data from parent)

- POST /api/youth-accounts/{youthAccountId}/fund-transfer → Transfer funds from parent account to youth account
- PUT /api/youth-accounts/{youthAccountId}/spending-limit → Configure weekly spending limits
- GET /api/youth-accounts/{youthAccountId}/transactions → Retrieve paginated transaction history with filtering

#### 1. YouthAccountApp (Root Page Component)

**Component Type:** Page Component
**Purpose:** Root application component managing global state, routing, and data orchestration for youth account management
**Parent:** None (Root)
**Children:** HeaderComponent, DashboardOverviewComponent, FundTransferComponent, ActivityTimelineComponent

**TypeScript Specification:**

```typescript
// types/youthAccount.types.ts
export interface YouthAccount {
  id: string;
  balance: number;
  currency: string;
  spendingLimit: number;
  weeklySpent: number;
  lastUpdated: string;
  parentAccountId: string;
}

export interface UserSession {
  userId: string;
  userName: string;
  userAvatar: string;
  isAuthenticated: boolean;
}

export interface DashboardData {
  youthAccount: YouthAccount;
  recentTransactions: Transaction[];
  parentAccounts: ParentAccount[];
}

// pages/YouthAccountApp/YouthAccountApp.tsx
import React, { useState, useEffect } from 'react';
import { HeaderComponent } from '../../components/layout/HeaderComponent';
import { DashboardOverviewComponent } from '../../components/feature/DashboardOverviewComponent';
import { FundTransferComponent } from '../../components/feature/FundTransferComponent';
import { ActivityTimelineComponent } from '../../components/feature/ActivityTimelineComponent';
import { useYouthAccount } from '../../hooks/useYouthAccount';
import { UserSession, DashboardData } from '../../types/youthAccount.types';
import styles from './YouthAccountApp.module.css';

interface YouthAccountAppProps {
  userId: string;
  youthAccountId: string;
}

interface YouthAccountAppState {
  currentView: 'dashboard' | 'transfer' | 'activity' | 'settings';
  userSession: UserSession;
  dashboardData: DashboardData | null;
  isLoading: boolean;
  error: string | null;
}

export const YouthAccountApp: React.FC<YouthAccountAppProps> = ({
  userId,
  youthAccountId
}) => {
  const [state, setState] = useState<YouthAccountAppState>({
    currentView: 'dashboard',
    userSession: {
      userId: '',
      userName: '',
      userAvatar: '',
      isAuthenticated: false
    },
    dashboardData: null,
    isLoading: true,
    error: null
  });

  const { fetchDashboardData, validateSession } = useYouthAccount();

  useEffect(() => {
    initializeApp();
  }, [userId, youthAccountId]);

  const initializeApp = async (): Promise<void> => {
    try {
      setState(prev => ({ ...prev, isLoading: true, error: null }));
      
      // Validate user session
      const sessionData = await validateSession(userId);
      
      // Fetch dashboard data
      const dashboardData = await fetchDashboardData(youthAccountId);
      
      setState(prev => ({
        ...prev,
        userSession: sessionData,
        dashboardData,
        isLoading: false
      }));
    } catch (error) {
      setState(prev => ({
        ...prev,
        error: error instanceof Error ? error.message : 'Failed to initialize app',
        isLoading: false
      }));
    }
  };

  const handleViewChange = (view: YouthAccountAppState['currentView']): void => {
    setState(prev => ({ ...prev, currentView: view }));
  };

  const handleSessionExpire = (): void => {
    setState(prev => ({
      ...prev,
      userSession: { ...prev.userSession, isAuthenticated: false }
    }));
    // Redirect to login
  };

  const handleDataRefresh = async (): Promise<void> => {
    try {
      const dashboardData = await fetchDashboardData(youthAccountId);
      setState(prev => ({ ...prev, dashboardData }));
    } catch (error) {
      setState(prev => ({
        ...prev,
        error: error instanceof Error ? error.message : 'Failed to refresh data'
      }));
    }
  };

  if (state.isLoading) {
    return <div className={styles.loading}>Loading youth account...</div>;
  }

  if (state.error) {
    return <div className={styles.error}>Error: {state.error}</div>;
  }

  if (!state.userSession.isAuthenticated) {
    return <div className={styles.unauthorized}>Please log in to continue</div>;
  }

  return (
    <div className={styles.youthAccountApp}>
      <HeaderComponent
        userName={state.userSession.userName}
        userAvatar={state.userSession.userAvatar}
        showSearch={false}
        onSearch={() => {}}
        onProfileClick={() => handleViewChange('settings')}
      />
      
      <div className={styles.mainLayout}>
        <nav className={styles.sidebar}>
          <button 
            className={`${styles.navItem} ${state.currentView === 'dashboard' ? styles.active : ''}`}
            onClick={() => handleViewChange('dashboard')}
          >
            Dashboard
          </button>
          <button 
            className={`${styles.navItem} ${state.currentView === 'transfer' ? styles.active : ''}`}
            onClick={() => handleViewChange('transfer')}
          >
            Add Funds
          </button>
          <button 
            className={`${styles.navItem} ${state.currentView === 'activity' ? styles.active : ''}`}
            onClick={() => handleViewChange('activity')}
          >
            Activity
          </button>
        </nav>
        
        <main className={styles.content}>
          {state.currentView === 'dashboard' && state.dashboardData && (
            <DashboardOverviewComponent
              accountBalance={state.dashboardData.youthAccount.balance}
              spendingLimit={state.dashboardData.youthAccount.spendingLimit}
              currency={state.dashboardData.youthAccount.currency}
              weeklySpent={state.dashboardData.youthAccount.weeklySpent}
              lastUpdated={state.dashboardData.youthAccount.lastUpdated}
              onRefresh={handleDataRefresh}
              onAddFunds={() => handleViewChange('transfer')}
              onSetLimit={() => {}}
            />
          )}
          
          {state.currentView === 'transfer' && state.dashboardData && (
            <FundTransferComponent
              sourceAccounts={state.dashboardData.parentAccounts}
              youthAccountId={youthAccountId}
              onTransferComplete={handleDataRefresh}
              onCancel={() => handleViewChange('dashboard')}
            />
          )}
          
          {state.currentView === 'activity' && (
            <ActivityTimelineComponent
              youthAccountId={youthAccountId}
              pageSize={20}
              currency={state.dashboardData?.youthAccount.currency || 'USD'}
            />
          )}
        </main>
      </div>
    </div>
  );
};
```

**TypeScript Specification:**

```typescript
// components/layout/HeaderComponent/HeaderComponent.tsx
import React, { useState } from 'react';
import { SearchBarComponent } from '@aava/play-core';
import { AvatarComponent } from '@aava/play-core';
import styles from './HeaderComponent.module.css';

interface HeaderComponentProps {
  userName: string;
  userAvatar: string;
  showSearch: boolean;
  onSearch: (query: string) => void;
  onProfileClick: () => void;
}

interface HeaderComponentState {
  searchQuery: string;
  showProfileMenu: boolean;
}

export const HeaderComponent: React.FC<HeaderComponentProps> = ({
  userName,
  userAvatar,
  showSearch,
  onSearch,
  onProfileClick
}) => {
  const [state, setState] = useState<HeaderComponentState>({
    searchQuery: '',
    showProfileMenu: false
  });

  const handleSearchChange = (query: string): void => {
    setState(prev => ({ ...prev, searchQuery: query }));
    onSearch(query);
  };

  const handleSearchClick = (): void => {
    onSearch(state.searchQuery);
  };

  const handleProfileClick = (): void => {
    setState(prev => ({ ...prev, showProfileMenu: !prev.showProfileMenu }));
    onProfileClick();
  };

  return (
```

**TypeScript Specification:**

```typescript
// components/feature/DashboardOverviewComponent/DashboardOverviewComponent.tsx
import React, { useState, useEffect } from 'react';
import { AccountBalanceCard } from '../AccountBalanceCard';
import { SpendingLimitCard } from '../SpendingLimitCard';
import { ButtonComponent } from '@aava/play-core';
import styles from './DashboardOverviewComponent.module.css';

interface DashboardOverviewComponentProps {
  accountBalance: number;
  spendingLimit: number;
  currency: string;
  weeklySpent: number;
  lastUpdated: string;
  onRefresh: () => Promise<void>;
  onAddFunds: () => void;
  onSetLimit: () => void;
}

interface DashboardOverviewComponentState {
  isLoading: boolean;
  refreshing: boolean;
  lastRefresh: string;
}

export const DashboardOverviewComponent: React.FC<DashboardOverviewComponentProps> = ({
  accountBalance,
  spendingLimit,
  currency,
  weeklySpent,
  lastUpdated,
  onRefresh,
  onAddFunds,
  onSetLimit
}) => {
  const [state, setState] = useState<DashboardOverviewComponentState>({
    isLoading: false,
    refreshing: false,
    lastRefresh: new Date().toISOString()
  });

  useEffect(() => {
    setState(prev => ({ ...prev, lastRefresh: lastUpdated }));
  }, [lastUpdated]);

  const handleRefresh = async (): Promise<void> => {
    try {
      setState(prev => ({ ...prev, refreshing: true }));
      await onRefresh();
      setState(prev => ({ 
        ...prev, 
        refreshing: false,
        lastRefresh: new Date().toISOString()
      }));
    } catch (error) {
      setState(prev => ({ ...prev, refreshing: false }));
      console.error('Failed to refresh dashboard:', error);
    }
  };

  const handleAddFunds = (): void => {
    onAddFunds();
  };

  const handleSetLimit = (): void => {
    onSetLimit();
  };

  const formatLastUpdated = (timestamp: string): string => {
    const date = new Date(timestamp);
    const now = new Date();
    const diffMinutes = Math.floor((now.getTime() - date.getTime()) / (1000 * 60));
    
    if (diffMinutes < 1) return 'Just now';
    if (diffMinutes < 60) return `${diffMinutes} minutes ago`;
    if (diffMinutes < 1440) return `${Math.floor(diffMinutes / 60)} hours ago`;
    return date.toLocaleDateString();
  };

  return (
    <div className={styles.dashboardOverview}>
      <div className={styles.header}>
        <h2 className={styles.title}>Youth Account Dashboard</h2>
        <div className={styles.headerActions}>
          <span className={styles.lastUpdated}>
            Last updated: {formatLastUpdated(state.lastRefresh)}
          </span>
          <ButtonComponent
            label={state.refreshing ? 'Refreshing...' : 'Refresh'}
            variant="secondary"
            size="small"
            disabled={state.refreshing}
            processing={state.refreshing}
            userClick={handleRefresh}
          />
        </div>
      </div>
      
      <div className={styles.cardsGrid}>
        <AccountBalanceCard
          balance={accountBalance}
          currency={currency}
          lastUpdated={lastUpdated}
          onRefresh={handleRefresh}
        />
        
        <SpendingLimitCard
          currentLimit={spendingLimit}
          currency={currency}
          weeklySpent={weeklySpent}
          onEditLimit={handleSetLimit}
          onSaveLimit={handleSetLimit}
        />
      </div>
      
      <div className={styles.quickActions}>
        <h3 className={styles.actionsTitle}>Quick Actions</h3>
        <div className={styles.actionsGrid}>
          <ButtonComponent
            label="Add Funds"
            variant="primary"
            size="large"
            disabled={false}
            processing={false}
            userClick={handleAddFunds}
          />
          <ButtonComponent
            label="Set Spending Limit"
            variant="secondary"
            size="large"
            disabled={false}
            processing={false}
            userClick={handleSetLimit}
          />
          <ButtonComponent
            label="View All Activity"
            variant="tertiary"
            size="large"
            disabled={false}
            processing={false}
            userClick={() => {}}
          />
        </div>
      </div>
      
      <div className={styles.summaryStats}>
        <div className={styles.stat}>
          <span className={styles.statLabel}>Available to Spend</span>
          <span className={styles.statValue}>
            {currency} {(spendingLimit - weeklySpent).toFixed(2)}
          </span>
        </div>
        <div className={styles.stat}>
          <span className={styles.statLabel}>Spending Progress</span>
          <div className={styles.progressBar}>
            <div 
              className={styles.progressFill}
              style={{ width: `${Math.min((weeklySpent / spendingLimit) * 100, 100)}%` }}
            />
          </div>
          <span className={styles.progressText}>
            {((weeklySpent / spendingLimit) * 100).toFixed(0)}% of limit used
          </span>
        </div>
      </div>
    </div>
  );
};
```

```typescript
// components/feature/AccountBalanceCard/AccountBalanceCard.tsx
import React, { useState } from 'react';
import { ButtonComponent } from '@aava/play-core';
import { formatCurrency } from '../../../utils/formatCurrency';
import { formatDate } from '../../../utils/formatDate';
import styles from './AccountBalanceCard.module.css';

interface AccountBalanceCardProps {
  balance: number;
  currency: string;
  lastUpdated: string;
  onRefresh: () => Promise<void>;
}

interface AccountBalanceCardState {
  isRefreshing: boolean;
  showDetails: boolean;
}

export const AccountBalanceCard: React.FC<AccountBalanceCardProps> = ({
  balance,
  currency,
  lastUpdated,
  onRefresh
}) => {
  const [state, setState] = useState<AccountBalanceCardState>({
    isRefreshing: false,
    showDetails: false
  });

  const handleRefresh = async (): Promise<void> => {
    try {
      setState(prev => ({ ...prev, isRefreshing: true }));
      await onRefresh();
    } catch (error) {
      console.error('Failed to refresh balance:', error);
    } finally {
      setState(prev => ({ ...prev, isRefreshing: false }));
    }
  };

  const toggleDetails = (): void => {
    setState(prev => ({ ...prev, showDetails: !prev.showDetails }));
  };

  const getBalanceStatus = (balance: number): 'healthy' | 'low' | 'critical' => {
    if (balance >= 50) return 'healthy';
    if (balance >= 10) return 'low';
    return 'critical';
  };

  const balanceStatus = getBalanceStatus(balance);
  const formattedBalance = formatCurrency(balance, currency);
  const formattedDate = formatDate(lastUpdated, 'relative');

  return (
    <div className={`${styles.balanceCard} ${styles[balanceStatus]}`}>
      <div className={styles.cardHeader}>
        <h3 className={styles.cardTitle}>Account Balance</h3>
        <ButtonComponent
          label={state.isRefreshing ? '↻' : '⟳'}
          variant="ghost"
          size="small"
          disabled={state.isRefreshing}
          processing={state.isRefreshing}
          userClick={handleRefresh}
        />
      </div>
      
      <div className={styles.balanceDisplay}>
        <span className={styles.currencySymbol}>{currency}</span>
        <span className={styles.balanceAmount}>{balance.toFixed(2)}</span>
      </div>
      
      <div className={styles.balanceInfo}>
        <div className={styles.statusIndicator}>
          <span className={`${styles.statusDot} ${styles[balanceStatus]}`} />
          <span className={styles.statusText}>
            {balanceStatus === 'healthy' && 'Good balance'}
            {balanceStatus === 'low' && 'Low balance'}
            {balanceStatus === 'critical' && 'Critical balance'}
          </span>
        </div>
        
        <button 
          className={styles.detailsToggle}
          onClick={toggleDetails}
          aria-expanded={state.showDetails}
        >
          {state.showDetails ? 'Hide details' : 'Show details'}
        </button>
      </div>
      
      {state.showDetails && (
        <div className={styles.balanceDetails}>
          <div className={styles.detailRow}>
            <span className={styles.detailLabel}>Last updated:</span>
            <span className={styles.detailValue}>{formattedDate}</span>
          </div>
          <div className={styles.detailRow}>
            <span className={styles.detailLabel}>Account type:</span>
            <span className={styles.detailValue}>Youth Savings</span>
          </div>
          <div className={styles.detailRow}>
            <span className={styles.detailLabel}>Available:</span>
            <span className={styles.detailValue}>{formattedBalance}</span>
          </div>
        </div>
      )}
      
      <div className={styles.cardFooter}>
        <small className={styles.disclaimer}>
          Balance updates in real-time during business hours
        </small>
      </div>
    </div>
  );
};
```

```typescript
// components/feature/SpendingLimitCard/SpendingLimitCard.tsx
import React, { useState, useEffect } from 'react';
import { ButtonComponent } from '@aava/play-core';
import { formatCurrency } from '../../../utils/formatCurrency';
import styles from './SpendingLimitCard.module.css';

interface SpendingLimitCardProps {
  currentLimit: number;
  currency: string;
  weeklySpent: number;
  onEditLimit: () => void;
  onSaveLimit: (newLimit: number) => Promise<void>;
}

interface SpendingLimitCardState {
  isEditing: boolean;
  editValue: string;
  isSaving: boolean;
  validationError: string | null;
}

export const SpendingLimitCard: React.FC<SpendingLimitCardProps> = ({
  currentLimit,
  currency,
  weeklySpent,
  onEditLimit,
  onSaveLimit
}) => {
  const [state, setState] = useState<SpendingLimitCardState>({
    isEditing: false,
    editValue: currentLimit.toString(),
    isSaving: false,
    validationError: null
  }));

  useEffect(() => {
    setState(prev => ({ ...prev, editValue: currentLimit.toString() }));
  }, [currentLimit]);

  const handleEditClick = (): void => {
    setState(prev => ({ 
      ...prev, 
      isEditing: true, 
      editValue: currentLimit.toString(),
      validationError: null 
    }));
    onEditLimit();
  };

  const handleCancelEdit = (): void => {
    setState(prev => ({ 
      ...prev, 
      isEditing: false, 
      editValue: currentLimit.toString(),
      validationError: null 
    }));
  };

  const handleSaveLimit = async (): Promise<void> => {
    const newLimit = parseFloat(state.editValue);
    
    // Validation
    if (isNaN(newLimit) || newLimit < 0) {
      setState(prev => ({ 
        ...prev, 
        validationError: 'Please enter a valid amount' 
      }));
      return;
    }
    
    if (newLimit > 1000) {
      setState(prev => ({ 
        ...prev, 
        validationError: 'Spending limit cannot exceed $1,000' 
      }));
      return;
    }
    
    try {
      setState(prev => ({ ...prev, isSaving: true, validationError: null }));
      await onSaveLimit(newLimit);
      setState(prev => ({ ...prev, isEditing: false, isSaving: false }));
    } catch (error) {
      setState(prev => ({ 
        ...prev, 
        isSaving: false,
        validationError: error instanceof Error ? error.message : 'Failed to save limit'
      }));
    }
  };

  const handleInputChange = (event: React.ChangeEvent<HTMLInputElement>): void => {
    const value = event.target.value;
    setState(prev => ({ ...prev, editValue: value, validationError: null }));
  };

  const getSpendingStatus = (): 'safe' | 'warning' | 'danger' => {
    const percentage = (weeklySpent / currentLimit) * 100;
    if (percentage >= 90) return 'danger';
    if (percentage >= 75) return 'warning';
    return 'safe';
  };

  const spendingPercentage = Math.min((weeklySpent / currentLimit) * 100, 100);
  const remainingAmount = Math.max(currentLimit - weeklySpent, 0);
  const spendingStatus = getSpendingStatus();

  return (
    <div className={`${styles.spendingCard} ${styles[spendingStatus]}`}>
      <div className={styles.cardHeader}>
        <h3 className={styles.cardTitle}>Weekly Spending Limit</h3>
        {!state.isEditing && (
          <ButtonComponent
            label="Edit"
            variant="ghost"
            size="small"
            disabled={false}
            processing={false}
            userClick={handleEditClick}
          />
        )}
      </div>
      
      <div className={styles.limitDisplay}>
        {state.isEditing ? (
          <div className={styles.editContainer}>
            <div className={styles.inputGroup}>
              <span className={styles.currencyPrefix}>{currency}</span>
              <input
                type="number"
                value={state.editValue}
                onChange={handleInputChange}
                className={styles.limitInput}
                placeholder="Enter limit"
                min="0"
                max="1000"
                step="0.01"
                autoFocus
              />
            </div>
            
            {state.validationError && (
              <div className={styles.errorMessage}>
                {state.validationError}
              </div>
            )}
            
            <div className={styles.editActions}>
              <ButtonComponent
                label="Cancel"
                variant="secondary"
                size="small"
                disabled={state.isSaving}
                processing={false}
                userClick={handleCancelEdit}
              />
              <ButtonComponent
                label={state.isSaving ? 'Saving...' : 'Save'}
                variant="primary"
                size="small"
                disabled={state.isSaving || !!state.validationError}
                processing={state.isSaving}
                userClick={handleSaveLimit}
              />
            </div>
          </div>
        ) : (
          <>
            <div className={styles.limitAmount}>
              <span className={styles.currencySymbol}>{currency}</span>
              <span className={styles.limitValue}>{currentLimit.toFixed(2)}</span>
              <span className={styles.limitPeriod}>/ week</span>
            </div>
            
            <div className={styles.spentAmount}>
              <span className={styles.spentLabel}>Spent this week:</span>
              <span className={styles.spentValue}>
                {formatCurrency(weeklySpent, currency)}
              </span>
            </div>
          </>
        )}
      </div>
      
      {!state.isEditing && (
        <>
          <div className={styles.progressSection}>
            <div className={styles.progressBar}>
              <div 
                className={`${styles.progressFill} ${styles[spendingStatus]}`}
                style={{ width: `${spendingPercentage}%` }}
              />
            </div>
            <div className={styles.progressLabels}>
              <span className={styles.progressText}>
                {spendingPercentage.toFixed(0)}% used
              </span>
              <span className={styles.remainingText}>
                {formatCurrency(remainingAmount, currency)} remaining
              </span>
            </div>
          </div>
          
          <div className={styles.statusMessage}>
            {spendingStatus === 'safe' && (
              <span className={styles.statusSafe}>
                ✓ You're on track with your spending
              </span>
            )}
            {spendingStatus === 'warning' && (
              <span className={styles.statusWarning}>
                ⚠ Approaching your spending limit
              </span>
            )}
            {spendingStatus === 'danger' && (
              <span className={styles.statusDanger}>
                ⚠ You've reached your spending limit
              </span>
            )}
          </div>
        </>
      )}
    </div>
  );
};
```

#### 6. FundTransferComponent (Feature Component)

**Component Type:** Feature Component
**Purpose:** Handle fund transfers from parent accounts to youth account with validation and confirmation
**Parent:** YouthAccountApp
**Children:** ButtonComponent (from library), ModalComponent, AlertComponent

**TypeScript Specification:**

```typescript
// types/transaction.types.ts
export interface ParentAccount {
  id: string;
  name: string;
  balance: number;
  accountNumber: string;
  type: 'checking' | 'savings';
}

export interface TransferRequest {
  sourceAccountId: string;
  targetAccountId: string;
  amount: number;
  description?: string;
}

export interface TransferResult {
  success: boolean;
  transactionId?: string;
  error?: string;
  newBalance?: number;
}

// components/feature/FundTransferComponent/FundTransferComponent.tsx
import React, { useState, useEffect } from 'react';
import { ButtonComponent } from '@aava/play-core';
import { ModalComponent } from '../../shared/ModalComponent';
import { AlertComponent } from '../../shared/AlertComponent';
import { useFundTransfer } from '../../../hooks/useFundTransfer';
import { ParentAccount, TransferRequest, TransferResult } from '../../../types/transaction.types';
import { formatCurrency } from '../../../utils/formatCurrency';
import styles from './FundTransferComponent.module.css';

interface FundTransferComponentProps {
  sourceAccounts: ParentAccount[];
  youthAccountId: string;
  onTransferComplete: () => Promise<void>;
  onCancel: () => void;
}

interface FundTransferComponentState {
  transferAmount: number | null;
  selectedSource: string;
  isProcessing: boolean;
  transferResult: TransferResult | null;
  validationErrors: {
    amount?: string;
    source?: string;
  };
  showConfirmation: boolean;
  description: string;
}

export const FundTransferComponent: React.FC<FundTransferComponentProps> = ({
  sourceAccounts,
  youthAccountId,
  onTransferComplete,
  onCancel
}) => {
  const [state, setState] = useState<FundTransferComponentState>({
    transferAmount: null,
    selectedSource: '',
    isProcessing: false,
    transferResult: null,
    validationErrors: {},
    showConfirmation: false,
    description: ''
  });

  const { submitTransfer, validateTransfer } = useFundTransfer();

  useEffect(() => {
    if (sourceAccounts.length > 0 && !state.selectedSource) {
      setState(prev => ({ ...prev, selectedSource: sourceAccounts[0].id }));
    }
  }, [sourceAccounts, state.selectedSource]);

  const handleAmountChange = (event: React.ChangeEvent<HTMLInputElement>): void => {
    const value = event.target.value;
    const amount = value ? parseFloat(value) : null;
    
    setState(prev => ({ 
      ...prev, 
      transferAmount: amount,
      validationErrors: { ...prev.validationErrors, amount: undefined }
    }));
  };

  const handleSourceChange = (event: React.ChangeEvent<HTMLSelectElement>): void => {
    setState(prev => ({ 
      ...prev, 
      selectedSource: event.target.value,
      validationErrors: { ...prev.validationErrors, source: undefined }
    }));
  };

  const handleDescriptionChange = (event: React.ChangeEvent<HTMLTextAreaElement>): void => {
    setState(prev => ({ ...prev, description: event.target.value }));
  };

  const validateForm = (): boolean => {
    const errors: typeof state.validationErrors = {};
    
    // Amount validation
    if (!state.transferAmount || state.transferAmount <= 0) {
      errors.amount = 'Please enter a valid amount';
    } else if (state.transferAmount > 500) {
      errors.amount = 'Transfer amount cannot exceed $500';
    } else if (state.transferAmount < 1) {
      errors.amount = 'Minimum transfer amount is $1';
    }
    
    // Source account validation
    if (!state.selectedSource) {
      errors.source = 'Please select a source account';
    } else {
      const sourceAccount = sourceAccounts.find(acc => acc.id === state.selectedSource);
      if (sourceAccount && state.transferAmount && sourceAccount.balance < state.transferAmount) {
        errors.amount = 'Insufficient funds in selected account';
      }
    }
    
    setState(prev => ({ ...prev, validationErrors: errors }));
    return Object.keys(errors).length === 0;
  };

  const handleTransferSubmit = async (): Promise<void> => {
    if (!validateForm()) return;
    
    setState(prev => ({ ...prev, showConfirmation: true }));
  };

  const handleConfirmTransfer = async (): Promise<void> => {
    if (!state.transferAmount || !state.selectedSource) return;
    
    const transferRequest: TransferRequest = {
      sourceAccountId: state.selectedSource,
      targetAccountId: youthAccountId,
      amount: state.transferAmount,
      description: state.description || 'Youth account funding'
    };
    
    try {
      setState(prev => ({ ...prev, isProcessing: true, showConfirmation: false }));
      
      const result = await submitTransfer(transferRequest);
      
      setState(prev => ({ ...prev, transferResult: result, isProcessing: false }));
      
      if (result.success) {
        await onTransferComplete();
        // Reset form after successful transfer
        setTimeout(() => {
          setState(prev => ({ 
            ...prev, 
            transferAmount: null, 
            description: '',
            transferResult: null 
          }));
        }, 3000);
      }
    } catch (error) {
      setState(prev => ({ 
        ...prev, 
        isProcessing: false,
        transferResult: {
          success: false,
          error: error instanceof Error ? error.message : 'Transfer failed'
        }
      }));
    }
  };

  const handleCancelTransfer = (): void => {
    setState(prev => ({ ...prev, showConfirmation: false }));
  };

  const handleCancel = (): void => {
    onCancel();
  };

  const selectedAccount = sourceAccounts.find(acc => acc.id === state.selectedSource);
  const isFormValid = state.transferAmount && state.selectedSource && Object.keys(state.validationErrors).length === 0;

  return (
    <div className={styles.fundTransfer}>
      <div className={styles.header}>
        <h2 className={styles.title}>Add Funds to Youth Account</h2>
        <p className={styles.subtitle}>
          Transfer money from your account to the youth account
        </p>
      </div>
      
      {state.transferResult && (
        <AlertComponent
          type={state.transferResult.success ? 'success' : 'error'}
          message={
            state.transferResult.success 
              ? `Transfer successful! Transaction ID: ${state.transferResult.transactionId}`
              : state.transferResult.error || 'Transfer failed'
          }
          onClose={() => setState(prev => ({ ...prev, transferResult: null }))}
        />
      )}
      
      <div className={styles.transferForm}>
        <div className={styles.formGroup}>
          <label htmlFor="sourceAccount" className={styles.label}>
            From Account
          </label>
          <select
            id="sourceAccount"
            value={state.selectedSource}
            onChange={handleSourceChange}
            className={`${styles.select} ${state.validationErrors.source ? styles.error : ''}`}
            disabled={state.isProcessing}
          >
            <option value="">Select source account</option>
            {sourceAccounts.map(account => (
              <option key={account.id} value={account.id}>
                {account.name} ({account.type}) - {formatCurrency(account.balance, 'USD')}
              </option>
            ))}
          </select>
          {state.validationErrors.source && (
            <span className={styles.errorText}>{state.validationErrors.source}</span>
          )}
        </div>
        
        <div className={styles.formGroup}>
          <label htmlFor="transferAmount" className={styles.label}>
            Transfer Amount
          </label>
          <div className={styles.amountInput}>
            <span className={styles.currencySymbol}>$</span>
            <input
              id="transferAmount"
              type="number"
              value={state.transferAmount || ''}
              onChange={handleAmountChange}
              placeholder="0.00"
              min="1"
              max="500"
              step="0.01"
              className={`${styles.input} ${state.validationErrors.amount ? styles.error : ''}`}
              disabled={state.isProcessing}
            />
          </div>
          {state.validationErrors.amount && (
            <span className={styles.errorText}>{state.validationErrors.amount}</span>
          )}
          
          {selectedAccount && (
            <div className={styles.accountInfo}>
              <span className={styles.availableBalance}>
                Available: {formatCurrency(selectedAccount.balance, 'USD')}
              </span>
            </div>
          )}
        </div>
        
        <div className={styles.formGroup}>
          <label htmlFor="description" className={styles.label}>
            Description (Optional)
          </label>
          <textarea
            id="description"
            value={state.description}
            onChange={handleDescriptionChange}
            placeholder="Add a note for this transfer"
            className={styles.textarea}
            disabled={state.isProcessing}
            rows={3}
          />
        </div>
        
        <div className={styles.formActions}>
          <ButtonComponent
            label="Cancel"
            variant="secondary"
            size="large"
            disabled={state.isProcessing}
            processing={false}
            userClick={handleCancel}
          />
          <ButtonComponent
            label={state.isProcessing ? 'Processing...' : 'Transfer Funds'}
            variant="primary"
            size="large"
            disabled={!isFormValid || state.isProcessing}
            processing={state.isProcessing}
            userClick={handleTransferSubmit}
          />
        </div>
      </div>
      
      {state.showConfirmation && (
        <ModalComponent
          isOpen={state.showConfirmation}
          onClose={handleCancelTransfer}
          title="Confirm Transfer"
        >
          <div className={styles.confirmationContent}>
            <p className={styles.confirmationText}>
              Are you sure you want to transfer{' '}
              <strong>{formatCurrency(state.transferAmount || 0, 'USD')}</strong>{' '}
              from <strong>{selectedAccount?.name}</strong> to the youth account?
            </p>
            
            {state.description && (
              <p className={styles.confirmationDescription}>
                Description: {state.description}
              </p>
            )}
            
            <div className={styles.confirmationActions}>
              <ButtonComponent
                label="Cancel"
                variant="secondary"
                size="medium"
                disabled={false}
                processing={false}
                userClick={handleCancelTransfer}
              />
              <ButtonComponent
                label="Confirm Transfer"
                variant="primary"
                size="medium"
                disabled={false}
                processing={false}
                userClick={handleConfirmTransfer}
              />
            </div>
          </div>
        </ModalComponent>
      )}
    </div>
  );
};
```

**API Integration:**
- POST /api/youth-accounts/{youthAccountId}/fund-transfer via submitTransfer hook
- Real-time validation via validateTransfer hook
- Parent account data received via props

**Error Handling:**
- Form validation errors → show inline error messages
- Insufficient funds → prevent submission, show error
- Network errors → show alert, allow retry
- API errors → show detailed error message

**Interaction Flows:**
1. Component mount → load source accounts → enable form
2. Form input → validate in real-time → show/hide errors
3. Submit click → validate form → show confirmation modal
4. Confirm transfer → process transfer → show result → refresh parent data
5. Cancel → close modal/component → return to dashboard

**Parent:** YouthAccountApp
**Children:** TransactionListComponent, SearchBarComponent (from library), ButtonComponent (from library)

**TypeScript Specification:**

```typescript
// types/transaction.types.ts
export interface Transaction {
  id: string;
  amount: number;
  currency: string;
  description: string;
  date: string;
  type: 'debit' | 'credit';
  category: string;
  merchant?: string;
  status: 'completed' | 'pending' | 'failed';
}

export interface TransactionFilter {
  dateFrom?: string;
  dateTo?: string;
  type?: 'debit' | 'credit' | 'all';
  category?: string;
  minAmount?: number;
  maxAmount?: number;
  searchQuery?: string;
}

// components/feature/ActivityTimelineComponent/ActivityTimelineComponent.tsx
import React, { useState, useEffect, useCallback } from 'react';
import { TransactionListComponent } from '../TransactionListComponent';
import { SearchBarComponent } from '@aava/play-core';
import { ButtonComponent } from '@aava/play-core';
import { useTransactionHistory } from '../../../hooks/useTransactionHistory';
import { Transaction, TransactionFilter } from '../../../types/transaction.types';
import { formatDate } from '../../../utils/formatDate';
import styles from './ActivityTimelineComponent.module.css';

interface ActivityTimelineComponentProps {
  youthAccountId: string;
  pageSize: number;
  currency: string;
}

interface ActivityTimelineComponentState {
  transactions: Transaction[];
  currentPage: number;
  totalPages: number;
  isLoading: boolean;
  filters: TransactionFilter;
  showFilters: boolean;
  error: string | null;
}

export const ActivityTimelineComponent: React.FC<ActivityTimelineComponentProps> = ({
  youthAccountId,
  pageSize,
  currency
}) => {
  const [state, setState] = useState<ActivityTimelineComponentState>({
    transactions: [],
    currentPage: 1,
    totalPages: 1,
    isLoading: true,
    filters: {
      type: 'all',
      searchQuery: ''
    },
    showFilters: false,
    error: null
  });

  const { fetchTransactions, exportTransactions } = useTransactionHistory();

  const loadTransactions = useCallback(async (page: number = 1, filters: TransactionFilter = state.filters): Promise<void> => {
    try {
      setState(prev => ({ ...prev, isLoading: true, error: null }));
      
      const response = await fetchTransactions(youthAccountId, {
        page,
        pageSize,
        ...filters
      });
      
      setState(prev => ({
        ...prev,
        transactions: response.transactions,
        currentPage: page,
        totalPages: response.totalPages,
        isLoading: false
      }));
    } catch (error) {
      setState(prev => ({
        ...prev,
        error: error instanceof Error ? error.message : 'Failed to load transactions',
        isLoading: false
      }));
    }
  }, [youthAccountId, pageSize, fetchTransactions, state.filters]);

  useEffect(() => {
    loadTransactions(1, state.filters);
  }, [youthAccountId]);

  const handlePageChange = (page: number): void => {
    if (page >= 1 && page <= state.totalPages && !state.isLoading) {
      loadTransactions(page, state.filters);
    }
  };

  const handleFilterChange = (newFilters: Partial<TransactionFilter>): void => {
    const updatedFilters = { ...state.filters, ...newFilters };
    setState(prev => ({ ...prev, filters: updatedFilters }));
    loadTransactions(1, updatedFilters);
  };

  const handleSearchChange = (query: string): void => {
    handleFilterChange({ searchQuery: query });
  };

  const handleSearchClick = (): void => {
    loadTransactions(1, state.filters);
  };

  const handleTransactionClick = (transaction: Transaction): void => {
    // Handle transaction detail view
    console.log('Transaction clicked:', transaction);
  };

  const handleExport = async (): Promise<void> => {
    try {
      await exportTransactions(youthAccountId, state.filters);
    } catch (error) {
      setState(prev => ({
        ...prev,
        error: error instanceof Error ? error.message : 'Failed to export transactions'
      }));
    }
  };

  const toggleFilters = (): void => {
    setState(prev => ({ ...prev, showFilters: !prev.showFilters }));
  };

  const clearFilters = (): void => {
    const defaultFilters: TransactionFilter = {
      type: 'all',
      searchQuery: ''
    };
    setState(prev => ({ ...prev, filters: defaultFilters }));
    loadTransactions(1, defaultFilters);
  };

  const hasActiveFilters = (): boolean => {
    return (
      state.filters.type !== 'all' ||
      !!state.filters.searchQuery ||
      !!state.filters.dateFrom ||
      !!state.filters.dateTo ||
      !!state.filters.category ||
      !!state.filters.minAmount ||
      !!state.filters.maxAmount
    );
  };

  return (
    <div className={styles.activityTimeline}>
      <div className={styles.header}>
        <h2 className={styles.title}>Transaction History</h2>
        <div className={styles.headerActions}>
          <SearchBarComponent
            label="Search transactions"
            disabled={state.isLoading}
            size="medium"
            searchClick={handleSearchClick}
            searchChange={handleSearchChange}
          />
          <ButtonComponent
            label="Filters"
            variant={hasActiveFilters() ? 'primary' : 'secondary'}
            size="medium"
            disabled={state.isLoading}
            processing={false}
            userClick={toggleFilters}
          />
          <ButtonComponent
            label="Export"
            variant="tertiary"
            size="medium"
            disabled={state.isLoading || state.transactions.length === 0}
            processing={false}
            userClick={handleExport}
          />
        </div>
      </div>
      
      {state.showFilters && (
        <div className={styles.filtersPanel}>
          <div className={styles.filtersGrid}>
            <div className={styles.filterGroup}>
              <label className={styles.filterLabel}>Transaction Type</label>
              <select
                value={state.filters.type || 'all'}
                onChange={(e) => handleFilterChange({ type: e.target.value as TransactionFilter['type'] })}
                className={styles.filterSelect}
              >
                <option value="all">All Transactions</option>
                <option value="credit">Money In</option>
                <option value="debit">Money Out</option>
              </select>
            </div>
            
            <div className={styles.filterGroup}>
              <label className={styles.filterLabel}>Date From</label>
              <input
                type="date"
                value={state.filters.dateFrom || ''}
                onChange={(e) => handleFilterChange({ dateFrom: e.target.value })}
                className={styles.filterInput}
              />
            </div>
            
            <div className={styles.filterGroup}>
              <label className={styles.filterLabel}>Date To</label>
              <input
                type="date"
                value={state.filters.dateTo || ''}
                onChange={(e) => handleFilterChange({ dateTo: e.target.value })}
                className={styles.filterInput}
              />
            </div>
            
            <div className={styles.filterGroup}>
              <label className={styles.filterLabel}>Min Amount</label>
              <input
                type="number"
                value={state.filters.minAmount || ''}
                onChange={(e) => handleFilterChange({ minAmount: parseFloat(e.target.value) || undefined })}
                placeholder="0.00"
                min="0"
                step="0.01"
                className={styles.filterInput}
              />
            </div>
            
            <div className={styles.filterGroup}>
              <label className={styles.filterLabel}>Max Amount</label>
              <input
                type="number"
                value={state.filters.maxAmount || ''}
                onChange={(e) => handleFilterChange({ maxAmount: parseFloat(e.target.value) || undefined })}
                placeholder="0.00"
                min="0"
                step="0.01"
                className={styles.filterInput}
              />
            </div>
          </div>
          
          <div className={styles.filterActions}>
            <ButtonComponent
              label="Clear Filters"
              variant="secondary"
              size="small"
              disabled={!hasActiveFilters()}
              processing={false}
              userClick={clearFilters}
            />
            <ButtonComponent
              label="Apply Filters"
              variant="primary"
              size="small"
              disabled={state.isLoading}
              processing={state.isLoading}
              userClick={() => loadTransactions(1, state.filters)}
            />
          </div>
        </div>
      )}
      
      {state.error && (
        <div className={styles.errorMessage}>
          <span className={styles.errorIcon}>⚠</span>
          <span>{state.error}</span>
          <ButtonComponent
            label="Retry"
            variant="secondary"
            size="small"
            disabled={false}
            processing={false}
            userClick={() => loadTransactions(state.currentPage, state.filters)}
          />
        </div>
      )}
      
      <div className={styles.transactionContainer}>
        {state.isLoading ? (
          <div className={styles.loading}>
            <div className={styles.loadingSpinner} />
            <span>Loading transactions...</span>
          </div>
        ) : state.transactions.length > 0 ? (
          <TransactionListComponent
            transactions={state.transactions}
            currency={currency}
            onTransactionSelect={handleTransactionClick}
          />
        ) : (
          <div className={styles.emptyState}>
            <div className={styles.emptyIcon}>📊</div>
            <h3 className={styles.emptyTitle}>No transactions found</h3>
            <p className={styles.emptyDescription}>
              {hasActiveFilters() 
                ? 'Try adjusting your filters to see more results'
                : 'Transactions will appear here once you start using your account'
              }
            </p>
            {hasActiveFilters() && (
              <ButtonComponent
                label="Clear Filters"
                variant="primary"
                size="medium"
                disabled={false}
                processing={false}
                userClick={clearFilters}
              />
            )}
          </div>
        )}
      </div>
      
      {state.totalPages > 1 && (
        <div className={styles.pagination}>
          <ButtonComponent
            label="Previous"
            variant="secondary"
            size="medium"
            disabled={state.currentPage <= 1 || state.isLoading}
            processing={false}
            userClick={() => handlePageChange(state.currentPage - 1)}
          />
          
          <div className={styles.pageInfo}>
            <span className={styles.pageNumbers}>
              {Array.from({ length: Math.min(5, state.totalPages) }, (_, i) => {
                const pageNum = Math.max(1, state.currentPage - 2) + i;
                if (pageNum <= state.totalPages) {
                  return (
                    <button
                      key={pageNum}
                      className={`${styles.pageButton} ${pageNum === state.currentPage ? styles.active : ''}`}
                      onClick={() => handlePageChange(pageNum)}
                      disabled={state.isLoading}
                    >
                      {pageNum}
                    </button>
                  );
                }
                return null;
              })}
            </span>
            <span className={styles.pageText}>
              Page {state.currentPage} of {state.totalPages}
            </span>
          </div>
          
          <ButtonComponent
            label="Next"
            variant="secondary"
            size="medium"
            disabled={state.currentPage >= state.totalPages || state.isLoading}
            processing={false}
            userClick={() => handlePageChange(state.currentPage + 1)}
          />
        </div>
      )}
    </div>
  );
};
```

**TypeScript Specification:**

```typescript
// components/feature/TransactionListComponent/TransactionListComponent.tsx
import React from 'react';
import { Transaction } from '../../../types/transaction.types';
import { formatCurrency } from '../../../utils/formatCurrency';
import { formatDate } from '../../../utils/formatDate';
import styles from './TransactionListComponent.module.css';

interface TransactionListComponentProps {
  transactions: Transaction[];
  currency: string;
  onTransactionSelect: (transaction: Transaction) => void;
}

export const TransactionListComponent: React.FC<TransactionListComponentProps> = ({
  transactions,
  currency,
  onTransactionSelect
}) => {
  const handleTransactionClick = (transaction: Transaction): void => {
    onTransactionSelect(transaction);
  };

  const getTransactionIcon = (type: Transaction['type'], category: string): string => {
    if (type === 'credit') {
      if (category === 'transfer') return '💰';
      if (category === 'allowance') return '🎁';
      return '💵';
    } else {
      if (category === 'food') return '🍔';
      if (category === 'entertainment') return '🎮';
      if (category === 'shopping') return '🛍️';
      if (category === 'transport') return '🚌';
      return '💳';
    }
  };

  const getStatusColor = (status: Transaction['status']): string => {
    switch (status) {
      case 'completed': return 'success';
      case 'pending': return 'warning';
      case 'failed': return 'error';
      default: return 'neutral';
    }
  };

  const formatTransactionDate = (date: string): { date: string; time: string } => {
    const transactionDate = new Date(date);
    const now = new Date();
    const diffDays = Math.floor((now.getTime() - transactionDate.getTime()) / (1000 * 60 * 60 * 24));
    
    let dateStr: string;
    if (diffDays === 0) {
      dateStr = 'Today';
    } else if (diffDays === 1) {
      dateStr = 'Yesterday';
    } else if (diffDays < 7) {
      dateStr = transactionDate.toLocaleDateString('en-US', { weekday: 'long' });
    } else {
      dateStr = transactionDate.toLocaleDateString('en-US', { 
        month: 'short', 
        day: 'numeric' 
      });
    }
    
    const timeStr = transactionDate.toLocaleTimeString('en-US', {
      hour: 'numeric',
      minute: '2-digit',
      hour12: true
    });
    
    return { date: dateStr, time: timeStr };
  };

  if (transactions.length === 0) {
    return (
      <div className={styles.emptyList}>
        <p className={styles.emptyMessage}>No transactions to display</p>
      </div>
    );
  }

  // Group transactions by date
  const groupedTransactions = transactions.reduce((groups, transaction) => {
    const date = new Date(transaction.date).toDateString();
    if (!groups[date]) {
      groups[date] = [];
    }
    groups[date].push(transaction);
    return groups;
  }, {} as Record<string, Transaction[]>);

  return (
    <div className={styles.transactionList}>
      {Object.entries(groupedTransactions).map(([dateKey, dayTransactions]) => {
        const groupDate = new Date(dateKey);
        const { date: displayDate } = formatTransactionDate(dayTransactions[0].date);
        
        return (
          <div key={dateKey} className={styles.transactionGroup}>
            <div className={styles.dateHeader}>
              <h3 className={styles.dateTitle}>{displayDate}</h3>
              <span className={styles.dateSubtitle}>
                {groupDate.toLocaleDateString('en-US', { 
                  year: 'numeric', 
                  month: 'long', 
                  day: 'numeric' 
                })}
              </span>
            </div>
            
            <div className={styles.transactionItems}>
              {dayTransactions.map((transaction) => {
                const { time } = formatTransactionDate(transaction.date);
                const statusColor = getStatusColor(transaction.status);
                const icon = getTransactionIcon(transaction.type, transaction.category);
                
                return (
                  <div
                    key={transaction.id}
                    className={`${styles.transactionItem} ${styles[transaction.type]} ${styles[statusColor]}`}
                    onClick={() => handleTransactionClick(transaction)}
                    role="button"
                    tabIndex={0}
                    onKeyDown={(e) => {
                      if (e.key === 'Enter' || e.key === ' ') {
                        handleTransactionClick(transaction);
                      }
                    }}
                  >
                    <div className={styles.transactionIcon}>
                      <span className={styles.iconEmoji}>{icon}</span>
                    </div>
                    
                    <div className={styles.transactionDetails}>
                      <div className={styles.transactionMain}>
                        <h4 className={styles.transactionDescription}>
                          {transaction.description}
                        </h4>
                        <div className={styles.transactionAmount}>
                          <span className={`${styles.amountValue} ${styles[transaction.type]}`}>
                            {transaction.type === 'credit' ? '+' : '-'}
                            {formatCurrency(transaction.amount, currency)}
                          </span>
                        </div>
                      </div>
                      
                      <div className={styles.transactionMeta}>
                        <div className={styles.metaLeft}>
                          {transaction.merchant && (
                            <span className={styles.merchant}>{transaction.merchant}</span>
                          )}
                          <span className={styles.category}>{transaction.category}</span>
                          <span className={styles.time}>{time}</span>
                        </div>
                        
                        <div className={styles.metaRight}>
                          <span className={`${styles.status} ${styles[statusColor]}`}>
                            {transaction.status === 'completed' && '✓'}
                            {transaction.status === 'pending' && '⏳'}
                            {transaction.status === 'failed' && '✗'}
                            {transaction.status.charAt(0).toUpperCase() + transaction.status.slice(1)}
                          </span>
                        </div>
                      </div>
                    </div>
                    
                    <div className={styles.transactionArrow}>
                      <span className={styles.arrowIcon}>›</span>
                    </div>
                  </div>
                );
              })}
            </div>
          </div>
        );
      })}
    </div>
  );
};
```
