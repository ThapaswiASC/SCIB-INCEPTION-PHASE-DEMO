# API Contract Outline
# Real-Time Task Creation Synchronization System

## Overview
This document outlines the API contract for the Real-Time Task Creation Synchronization System, including REST endpoints, WebSocket protocols, request/response schemas, and error handling specifications.

## Base Configuration
- **Base URL**: `https://api.scib.platform.com/v1`
- **Authentication**: Bearer Token (JWT)
- **Content-Type**: `application/json`
- **API Version**: v1

---

## REST API Endpoints

### 1. Single Task Creation

#### Endpoint
```
POST /api/boards/{boardId}/tasks
```

#### Path Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| boardId | UUID | Yes | Unique identifier of the board |

#### Request Headers
```
Authorization: Bearer {jwt_token}
Content-Type: application/json
```

#### Request Schema
```json
{
  "title": {
    "type": "string",
    "required": true,
    "maxLength": 200,
    "minLength": 1,
    "description": "Task title, must be unique within the board"
  },
  "description": {
    "type": "string",
    "required": false,
    "maxLength": 2000,
    "description": "Detailed task description"
  },
  "assigneeId": {
    "type": "string",
    "format": "uuid",
    "required": false,
    "description": "UUID of the assigned user"
  },
  "priority": {
    "type": "string",
    "enum": ["LOW", "MEDIUM", "HIGH", "CRITICAL"],
    "required": true,
    "description": "Task priority level"
  },
  "dueDate": {
    "type": "string",
    "format": "date",
    "required": false,
    "description": "Task due date (must be future date)"
  },
  "columnId": {
    "type": "string",
    "format": "uuid",
    "required": true,
    "description": "UUID of the target column"
  },
  "tags": {
    "type": "array",
    "items": {
      "type": "string",
      "maxLength": 50
    },
    "maxItems": 10,
    "required": false,
    "description": "Array of task tags"
  }
}
```

#### Error Handling

**400 Bad Request - Validation Errors**
```json
{
  "error": {
    "code": "VALIDATION_FAILED",
    "message": "Request validation failed",
    "details": [
      {
        "field": "title",
        "code": "REQUIRED",
        "message": "Title is required"
      },
      {
        "field": "title",
        "code": "MAX_LENGTH_EXCEEDED",
        "message": "Title must not exceed 200 characters"
      },
      {
        "field": "dueDate",
        "code": "INVALID_DATE",
        "message": "Due date must be a future date"
      }
    ],
    "timestamp": "2024-01-15T10:30:00Z",
    "path": "/api/boards/123/tasks"
  }
}
```

**409 Conflict - Duplicate Title**
```json
{
  "error": {
    "code": "DUPLICATE_TASK_TITLE",
    "message": "A task with this title already exists in the board",
    "details": {
      "field": "title",
      "value": "Implement user authentication",
      "existingTaskId": "existing-task-uuid"
    },
    "timestamp": "2024-01-15T10:30:00Z",
    "path": "/api/boards/123/tasks"
  }
}
```

**403 Forbidden - Insufficient Permissions**
```json
{
  "error": {
    "code": "INSUFFICIENT_PERMISSIONS",
    "message": "User does not have permission to create tasks in this board",
    "timestamp": "2024-01-15T10:30:00Z",
    "path": "/api/boards/123/tasks"
  }
}
```

**404 Not Found - Board/Column Not Found**
```json
{
  "error": {
    "code": "RESOURCE_NOT_FOUND",
    "message": "Board or column not found",
    "details": {
      "resourceType": "board",
      "resourceId": "invalid-board-id"
    },
    "timestamp": "2024-01-15T10:30:00Z",
    "path": "/api/boards/123/tasks"
  }
}
```

**500 Internal Server Error**
```json
{
  "error": {
    "code": "INTERNAL_SERVER_ERROR",
    "message": "An unexpected error occurred while processing the request",
    "correlationId": "abc123-def456-ghi789",
    "timestamp": "2024-01-15T10:30:00Z",
    "path": "/api/boards/123/tasks"
  }
}
```

### 2. Bulk Task Creation

#### Endpoint
```
POST /api/boards/{boardId}/tasks/batch
```

#### Path Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| boardId | UUID | Yes | Unique identifier of the board |

#### Request Headers
```
Authorization: Bearer {jwt_token}
Content-Type: application/json
```

#### Request Schema
```json
{
  "tasks": {
    "type": "array",
    "items": {
      "$ref": "#/components/schemas/TaskCreationRequest"
    },
    "minItems": 1,
    "maxItems": 1000,
    "description": "Array of task creation requests"
  },
  "options": {
    "type": "object",
    "properties": {
      "async": {
        "type": "boolean",
        "default": false,
        "description": "Process batch asynchronously (auto-enabled for >100 tasks)"
      },
      "validateOnly": {
        "type": "boolean",
        "default": false,
        "description": "Only validate tasks without creating them"
      }
    }
  }
}
```

#### Request Example
```json
{
  "tasks": [
    {
      "title": "Setup CI/CD pipeline",
      "description": "Configure automated build and deployment",
      "priority": "HIGH",
      "columnId": "987fcdeb-51a2-43d1-b789-123456789abc",
      "tags": ["devops", "automation"]
    },
    {
      "title": "Write unit tests",
      "description": "Add comprehensive test coverage",
      "priority": "MEDIUM",
      "columnId": "987fcdeb-51a2-43d1-b789-123456789abc",
      "assigneeId": "123e4567-e89b-12d3-a456-426614174000",
      "tags": ["testing", "quality"]
    }
  ],
  "options": {
    "async": false,
    "validateOnly": false
  }
}
```

#### Response Schema (Success - 201 Created)
```json
{
  "batchId": {
    "type": "string",
    "format": "uuid",
    "description": "Batch operation identifier"
  },
  "status": {

  },
  "summary": {
    "type": "object",
    "properties": {
      "totalRequested": {"type": "integer"},
      "successfullyCreated": {"type": "integer"},
      "failed": {"type": "integer"},
      "validationErrors": {"type": "integer"}
    }
  },
  "tasks": {
    "type": "array",
    "items": {
      "$ref": "#/components/schemas/TaskResponse"
    },
    "description": "Array of created tasks (if synchronous)"
  },
  "errors": {
    "type": "array",
    "items": {
      "type": "object",
      "properties": {
        "index": {"type": "integer"},
```

## WebSocket Protocol

### Connection
- **URL**: `wss://api.scib.platform.com/ws/boards/{boardId}`
- **Authentication**: JWT token in connection headers or query parameter
- **Heartbeat**: 30-second intervals

### Message Format
All WebSocket messages follow this structure:
```json
{
  "type": "string",
  "timestamp": "ISO 8601 datetime",
  "boardId": "UUID",
  "data": "object"
}
```

## Rate Limiting

### API Rate Limits
- **Single Task Creation**: 100 requests per minute per user
- **Bulk Task Creation**: 10 requests per minute per user
- **Validation Endpoint**: 200 requests per minute per user
- **WebSocket Connections**: 10 concurrent connections per user

### Rate Limit Headers
```
X-RateLimit-Limit: 100
X-RateLimit-Remaining: 95
X-RateLimit-Reset: 1642248600
X-RateLimit-Window: 60
```

### Rate Limit Exceeded Response (429 Too Many Requests)
```json
{
  "error": {
    "code": "RATE_LIMIT_EXCEEDED",
    "message": "Rate limit exceeded. Please try again later.",
    "details": {
      "limit": 100,
      "window": 60,
      "retryAfter": 30
    },
    "timestamp": "2024-01-15T10:30:00Z"
  }
}
```

## Performance Guarantees

### Response Time SLAs
- **Single Task Creation**: < 200ms (95th percentile)
- **Bulk Task Creation (≤100 tasks)**: < 2s (95th percentile)
- **Task Validation**: < 100ms (95th percentile)
- **WebSocket Message Delivery**: < 50ms (95th percentile)

### Synchronization Guarantees
- **Real-time Sync**: Tasks appear on all connected clients within 500ms
- **Message Ordering**: Tasks created in sequence maintain order across all clients
- **Delivery Guarantee**: At-least-once delivery for WebSocket messages

## Error Codes Reference

### Validation Error Codes
| Code | Description | HTTP Status |
|------|-------------|-------------|
| REQUIRED | Field is required | 400 |
| MAX_LENGTH_EXCEEDED | Field exceeds maximum length | 400 |
| MIN_LENGTH_NOT_MET | Field below minimum length | 400 |
| INVALID_FORMAT | Field format is invalid | 400 |
| INVALID_ENUM_VALUE | Value not in allowed enum | 400 |
| INVALID_DATE | Date format or value invalid | 400 |
| FUTURE_DATE_REQUIRED | Date must be in the future | 400 |
| DUPLICATE_TASK_TITLE | Title already exists in board | 409 |
| INVALID_ASSIGNEE | Assignee not found or no access | 400 |
| INVALID_COLUMN | Column not found in board | 400 |

### System Error Codes
| Code | Description | HTTP Status |
|------|-------------|-------------|
| INSUFFICIENT_PERMISSIONS | User lacks required permissions | 403 |
| RESOURCE_NOT_FOUND | Requested resource not found | 404 |
| RATE_LIMIT_EXCEEDED | Too many requests | 429 |
| INTERNAL_SERVER_ERROR | Unexpected server error | 500 |
| SERVICE_UNAVAILABLE | Service temporarily unavailable | 503 |
| BATCH_SIZE_EXCEEDED | Batch exceeds maximum size | 400 |
| ASYNC_PROCESSING_FAILED | Asynchronous processing error | 500 |
