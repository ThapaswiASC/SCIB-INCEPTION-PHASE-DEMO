# API Contract Outline
# Task Creation Synchronization System
# Version 1.0
# Date: 2024

## Overview
This document outlines the API contracts for the Task Creation Synchronization System, providing detailed specifications for REST endpoints, WebSocket interfaces, request/response schemas, and error handling patterns.

## Base Configuration
- **Base URL**: `https://api.scib.com/v1`
- **Authentication**: Bearer Token (JWT)
- **Content-Type**: `application/json`
- **API Version**: v1
- **Rate Limiting**: 100 requests per minute per user

## REST API Endpoints

### 1. Single Task Creation

#### Endpoint Details
- **Method**: `POST`
- **Path**: `/api/boards/{boardId}/tasks`
- **Description**: Creates a single task and broadcasts to all connected users
- **Authentication**: Required
- **Rate Limit**: 10 requests per minute per user

#### Path Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| boardId | UUID | Yes | Unique identifier of the board |

#### Request Headers
```http
Authorization: Bearer {jwt_token}
Content-Type: application/json
X-Request-ID: {uuid} (optional)
```

### 2. Bulk Task Creation

#### Endpoint Details
- **Method**: `POST`
- **Path**: `/api/boards/{boardId}/tasks/batch`
- **Description**: Creates multiple tasks in a single operation with atomic transaction
- **Authentication**: Required
- **Rate Limit**: 2 requests per minute per user
- **Max Batch Size**: 1000 tasks

#### Path Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| boardId | UUID | Yes | Unique identifier of the board |

#### Request Headers
```http
Authorization: Bearer {jwt_token}
Content-Type: application/json
X-Request-ID: {uuid} (optional)
```

#### Request Schema
```json
{
  "tasks": {
    "type": "array",
    "items": {
      "$ref": "#/components/schemas/TaskCreationRequest"
    },
    "minItems": 1,
    "maxItems": 1000,
    "description": "Array of task creation requests"
  },
  "options": {
    "type": "object",
    "properties": {
      "validateAll": {
        "type": "boolean",
        "default": true,
        "description": "Validate all tasks before creating any"
      },
      "stopOnError": {
        "type": "boolean",
        "default": true,
        "description": "Stop processing on first validation error"
      },
      "async": {
        "type": "boolean",
        "default": false,
        "description": "Process batch asynchronously for large batches"
      }
    }
  }
}
```

#### Request Example
```json
{
  "tasks": [
    {
      "title": "Setup CI/CD pipeline",
      "description": "Configure automated build and deployment",
```

- **URL**: `wss://api.scib.com/ws/boards/{boardId}`

## Error Codes Reference

### Validation Error Codes
| Code | Description | HTTP Status |
|------|-------------|-------------|
| REQUIRED_FIELD_MISSING | Required field is missing or null | 400 |
| INVALID_FIELD_LENGTH | Field length exceeds maximum or below minimum | 400 |
| INVALID_ENUM_VALUE | Field value is not in allowed enum values | 400 |
| INVALID_DATE_FORMAT | Date field is not in ISO 8601 format | 400 |
| INVALID_UUID_FORMAT | UUID field is not in valid UUID format | 400 |
| DUPLICATE_VALUE | Field value violates uniqueness constraint | 409 |
| INVALID_REFERENCE | Referenced entity does not exist | 400 |
| FUTURE_DATE_REQUIRED | Date field must be in the future | 400 |

### Authentication/Authorization Error Codes
| Code | Description | HTTP Status |
|------|-------------|-------------|
| AUTHENTICATION_REQUIRED | No authentication token provided | 401 |
| INVALID_TOKEN | Authentication token is invalid or malformed | 401 |
| EXPIRED_TOKEN | Authentication token has expired | 401 |
| INSUFFICIENT_PERMISSIONS | User lacks required permissions | 403 |
| BOARD_ACCESS_DENIED | User does not have access to the board | 403 |

### Resource Error Codes
| Code | Description | HTTP Status |
|------|-------------|-------------|
| BOARD_NOT_FOUND | Specified board does not exist | 404 |
| USER_NOT_FOUND | Referenced user does not exist | 404 |
| COLUMN_NOT_FOUND | Referenced column does not exist | 404 |
| BATCH_NOT_FOUND | Specified batch operation not found | 404 |

### Rate Limiting Error Codes
| Code | Description | HTTP Status |
|------|-------------|-------------|
| RATE_LIMIT_EXCEEDED | API rate limit exceeded | 429 |
| CONCURRENT_LIMIT_EXCEEDED | Too many concurrent requests | 429 |
| BATCH_SIZE_LIMIT_EXCEEDED | Batch size exceeds maximum allowed | 413 |

### System Error Codes
| Code | Description | HTTP Status |
|------|-------------|-------------|
| INTERNAL_SERVER_ERROR | Unexpected server error | 500 |
| DATABASE_CONNECTION_ERROR | Database connection failed | 503 |
| MESSAGE_BROKER_ERROR | Message broker unavailable | 503 |
| SERVICE_UNAVAILABLE | Service temporarily unavailable | 503 |

#### Task Creation Service
```typescript
import { Injectable } from '@angular/core';
import { HttpClient, HttpHeaders } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class TaskService {
  private baseUrl = 'https://api.scib.com/v1';

  constructor(private http: HttpClient) {}

  createTask(boardId: string, task: TaskCreationRequest): Observable<Task> {
    const headers = new HttpHeaders({
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${this.getAuthToken()}`
    });

    return this.http.post<Task>(
      `${this.baseUrl}/api/boards/${boardId}/tasks`,
      task,
      { headers }
    );
  }

  createTasksBatch(boardId: string, batchRequest: BatchTaskCreationRequest): Observable<BatchCreationResponse> {
    const headers = new HttpHeaders({
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${this.getAuthToken()}`
    });

    return this.http.post<BatchCreationResponse>(
      `${this.baseUrl}/api/boards/${boardId}/tasks/batch`,
      batchRequest,
      { headers }
    );
  }

  private getAuthToken(): string {
    return localStorage.getItem('authToken') || '';
  }
}
```

#### WebSocket Service
```typescript
import { Injectable } from '@angular/core';
import { Subject, Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class WebSocketService {
  private socket: WebSocket | null = null;
  private messageSubject = new Subject<any>();

  connect(boardId: string): Observable<any> {
    const token = localStorage.getItem('authToken');
    const wsUrl = `wss://api.scib.com/ws/boards/${boardId}?token=${token}`;
    
    this.socket = new WebSocket(wsUrl);
    
    this.socket.onmessage = (event) => {
      const message = JSON.parse(event.data);
      this.messageSubject.next(message);
    };

    this.socket.onopen = () => {
      this.subscribe(boardId, ['TASK_CREATED', 'TASK_UPDATED', 'TASK_DELETED']);
    };

    return this.messageSubject.asObservable();
  }

  private subscribe(boardId: string, events: string[]): void {
    if (this.socket && this.socket.readyState === WebSocket.OPEN) {
      this.socket.send(JSON.stringify({
        action: 'SUBSCRIBE',
        boardId: boardId,
        events: events
      }));
    }
  }

  disconnect(): void {
    if (this.socket) {
      this.socket.close();
      this.socket = null;
    }
  }
}
```

### React Example

#### Task Creation Hook
```typescript
import { useState, useCallback } from 'react';

interface UseTaskCreationResult {
  createTask: (boardId: string, task: TaskCreationRequest) => Promise<Task>;
  isLoading: boolean;
  error: string | null;
}

export const useTaskCreation = (): UseTaskCreationResult => {
  const [isLoading, setIsLoading] = useState(false);
  const [error, setError] = useState<string | null>(null);

  const createTask = useCallback(async (boardId: string, task: TaskCreationRequest): Promise<Task> => {
    setIsLoading(true);
    setError(null);

    try {
      const response = await fetch(`https://api.scib.com/v1/api/boards/${boardId}/tasks`, {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'Authorization': `Bearer ${localStorage.getItem('authToken')}`
        },
        body: JSON.stringify(task)
      });

      if (!response.ok) {
        const errorData = await response.json();
        throw new Error(errorData.error.message);
      }

      return await response.json();
    } catch (err) {
      const errorMessage = err instanceof Error ? err.message : 'An unexpected error occurred';
      setError(errorMessage);
      throw err;
    } finally {
      setIsLoading(false);
    }
  }, []);

  return { createTask, isLoading, error };
};
```
